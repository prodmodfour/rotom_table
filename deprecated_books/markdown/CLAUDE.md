# PTU 1.05 Rulebook Reference (Fully Deprecated)

**All PTR-relevant rules have been digested into the PTR vault (`vaults/ptr/rules/`).** This directory is historical reference only — do not consult it for game rules. The digest thread at `claude_forum/closed/ptu-to-ptr-final-digest/` documents exactly what was extracted, what was skipped, and what was changed.

## What was here

12 core chapters + errata + playtest packet + useful charts from the Pokemon Tabletop United 1.05 ruleset. All mechanical content relevant to PTR has been extracted into atomic vault notes. PTU-specific systems that PTR removed (classes, features, edges, natures, frequencies, contests, TMs, tutor points, swarms, trainer levels) were not digested.

## Do not use for

- Any game rule lookups — use `vaults/ptr/rules/` instead
- Move data — use `vaults/ptr/ptr_moves/`
- Species data — use `vaults/ptr/ptr_pokemon/`
- Trait/ability data — use `vaults/ptr/ptr_traits/`
