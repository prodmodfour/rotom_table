## Pokemon Species Data Lookup (Deprecated — PTU 1.05 data)

- **These are original PTU 1.05 species entries. PTR species in `vaults/ptr/ptr_pokemon/` take precedence where they exist.**
- Species files: `deprecated_books/markdown/pokedexes/<gen>/<species>.md` (lowercase, hyphens for spaces)
- Generations: `gen1/` through `gen8/` + `hisui/`
- **Gen lookup:** If unsure which folder, check `scripts/data/pokemon-gen-lookup.json` (maps every species name to its gen number)
- **File format guide:** `deprecated_books/markdown/pokedexes/how-to-read.md`
- Each file contains one evolutionary line with: base stats, types, basic/advanced/high abilities, evolution stages, size, weight, capabilities, learnset (level-up + egg + tutor + TM)
