## Summary

The character POST and PUT API endpoints do not enforce a maximum trainer level of 50. A GM can set level > 50 via direct API calls or edge-case UI interactions.

## Suggested Fix

Add server-side validation on character create/update to reject level values > 50.

## Impact

LOW in practice (GM-only action, unlikely accidental), but violates PTU core rules.
