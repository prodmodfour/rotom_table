## Summary

The `/gm/encounter-tables` page returns HTTP 500 because it references `/icons/phosphor/upload-simple.svg` which does not exist in the public directory. This blocks the encounter tables list page and all child routes.

## Problem

The page template imports or references the `upload-simple` Phosphor icon as an SVG file, but the icon was never added to `app/public/icons/phosphor/`.

## Suggested Fix

Add the `upload-simple.svg` icon file from the Phosphor Icons package to `app/public/icons/phosphor/`.

## Impact

- 3 encounter-tables capabilities directly blocked (C023, C024, C047)
- The encounter tables list page (`/gm/encounter-tables`) is completely inaccessible
- Also noted by vtt-grid browser audit as interfering with SPA navigation via Vite HMR errors
