# bug-041: Remove Whirlwind references from Force Switch UI per decree-034

## Summary

CombatantCard Force Switch button tooltip and comment reference "Roar/Whirlwind" as forced switches. decree-034 explicitly rules that Whirlwind is a push, NOT a forced switch. Whirlwind references must be removed.

Additionally, the design spec (design-pokemon-switching-001 Section I) references Whirlwind and Dragon Tail as forced switches. Whirlwind should be removed per decree-034. Dragon Tail and Circle Throw need individual PTU text analysis (not yet covered by any decree).

## Affected Files

- `app/components/encounter/CombatantCard.vue` — Force Switch button title/tooltip attribute
- `artifacts/designs/design-pokemon-switching-001/spec-p1.md` — Section I forced switch references

## Suggested Fix

1. Change "Roar/Whirlwind" to just "Roar" in CombatantCard.vue
2. Update design spec Section I to remove Whirlwind
3. Add a note that Dragon Tail/Circle Throw forced switch behavior is TBD (no decree yet)

## Impact

decree-034 compliance. Cosmetic/text-only change — no logic affected.
