## Summary

Dim Cave description says "Negated by Darkvision" and Dark Cave says "Negated by Blindsense", which is misleadingly narrow. Per RAW and rules-review-303 RULING-1, both Darkvision and Blindsense negate both tiers of darkness penalties. A GM reading only the description might believe Darkvision is insufficient for Dark Cave.

## Suggested Fix

Update both preset descriptions to list both capabilities: "Negated by Darkvision or Blindsense".

## Impact

Low — cosmetic but prevents GM confusion about which capabilities apply to which darkness tiers.
