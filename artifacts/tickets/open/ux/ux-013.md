# ux-013: Stacked bonus Skill Edge rank-up display shows incorrect progression in LevelUpSummary

## Summary

When multiple bonus Skill Edges (e.g., from levels 2, 6, and 12 in a multi-level jump) raise the same skill, `skillRankUpDetails` in LevelUpSummary.vue shows each rank-up independently from the character's base rank. For example, raising Acrobatics (Untrained) at L2 and again at L6 shows two entries both displaying "Untrained -> Novice" instead of "Untrained -> Novice" and "Novice -> Adept".

## Affected Files

- `app/components/levelup/LevelUpSummary.vue` (lines 211-222)

## Impact

Cosmetic display issue only. The actual `buildUpdatePayload` correctly applies stacked rank-ups sequentially — the data written to the database is correct. Only the summary preview is potentially misleading in the multi-jump stacking scenario.

## Suggested Fix

Compute cumulative effective rank for each skill when building `skillRankUpDetails`, accounting for prior rank-ups in the same level-up session.
