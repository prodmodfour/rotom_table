# Changes
	- Removing Body, Mind, Spirit categories
	- Removing ranks
	- Removing d6 system
		- Now 1d20 + Modifier
	- Features and Edges no longer exist so they are no longer prereqs
	- Modifiers are gained through Traits or Circumstance Bonuses
	- Humans and Pokemon have the same Skill List
