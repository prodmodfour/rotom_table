Ghost-type Pokemon ignore movement-restricting terrain and conditions (Stuck, Trapped) per PTU type rules.

The server enforces this via [[server-enforcement-with-gm-override]], so the GM can override for special scenarios where Ghost immunity is suppressed (e.g. a Gravity field or custom encounter rule). By default, Ghost-type combatants pass through terrain and status checks that would restrict movement for other types.

This is one instance of the broader [[type-grants-status-immunity]] pattern, where a Pokemon's type automatically exempts it from certain conditions.
