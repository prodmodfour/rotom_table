Elevation values set on grid cells persist across sessions as part of the encounter's saved state.

The [[the-table-as-shared-space|group view]] reflects the same elevation data that the GM configured. Elevation is not a transient visual — it is game state that affects movement costs, line of sight, and attack ranges.

This persistence is part of [[grid-mode-is-encounter-identity]], where the grid's spatial data (including elevation) defines the encounter. The [[elevation-system]] governs how elevation values interact with other mechanics.
