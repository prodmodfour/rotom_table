Trainer capabilities (Overland speed, swimming, jumping distances, etc.) are computed from skill ranks rather than set independently.

When a skill rank changes, all dependent capabilities update automatically. The trainer never manually edits a capability value that is derived from a skill — the app handles this as part of [[automate-routine-bookkeeping]].

This means the [[trainer-capabilities-field]] displays computed values sourced from [[trainer-skill-definitions]], not freeform inputs.
