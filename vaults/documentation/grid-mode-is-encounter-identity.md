# Grid Mode Is Encounter Identity

The choice of grid presentation (2D top-down vs isometric) is part of an encounter's persistent identity. An encounter set up in isometric mode should remain isometric across reloads and sessions.

This is the same principle as persisting grid dimensions, background image, and terrain — the visual setup is part of the encounter's prepared state. A GM who spends time configuring an isometric battlefield should not have to reconfigure it every time the encounter is loaded.
