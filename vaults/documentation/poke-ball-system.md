# Poke Ball System

25 PTR ball types with capture modifiers. See [[capture-rate-formula]] for the base rate calculation that ball modifiers adjust.

## Ball Categories

Balls are organized into categories: basic, apricorn, special, and safari. Each ball has a base modifier and optionally a conditional modifier that applies only when specific conditions are met.

## Conditional Balls

13 balls have conditional modifiers: Timer, Quick, Level, Heavy, Fast, Love, Net, Dusk, Moon, Lure, Repeat, Nest, Dive. The system evaluates whether the condition is met and applies the additional modifier accordingly.

## Post-Capture Effects

Some balls apply effects after a successful capture:
- **Heal Ball:** fully restore HP
- **Friend Ball:** +1 loyalty
- **Luxury Ball:** raised happiness

## Capture Context

The GM can set context toggles for conditional evaluation: whether the target was baited, whether the environment is dark or low-light, and whether the capture is underwater or underground.
