# Replace Magic Number with Symbolic Constant

An [[organizing-data-techniques|organizing data]] [[refactoring-techniques|technique]]. When code uses a number whose meaning isn't obvious, replace it with a named constant.

The constant name documents the number's purpose and makes future changes easier — update one definition instead of hunting for every occurrence.
