# Comments Smell

A [[dispensable-smells|dispensable]] [[code-smells|smell]]. A method filled with explanatory comments. Excessive comments often indicate the code itself isn't self-explanatory — the comments are compensating for poor naming, complex structure, or tangled logic.

The best comment is a good name for a method or class. If you feel a code fragment needs a comment, try refactoring it so the comment becomes unnecessary.
