# UML Behavioral View

A [[uml-view|view]] within [[uml-modeling-it-systems|IT system modeling]] that documents dynamic object behavior using [[uml-statechart-diagram|statechart diagrams]]. Each diagram represents an individual object's [[uml-object-lifecycle|lifecycle]] and its [[uml-dynamic-business-rule|dynamic business rules]].

Not every class needs a statechart diagram. Focus on those that either contain many or important business rules, or describe important objects.
