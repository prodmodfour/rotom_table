# UML System Integration Interface

The fundamental building block of [[uml-modeling-system-integration|system integration]]. An interface enables one IT system (sender) to transmit information to another (receiver). Any system can function as sender, receiver, or both simultaneously.

Connected IT systems exchange [[uml-system-integration-message|messages]] via their interfaces. For the receiving system, each message constitutes an event requiring response.
