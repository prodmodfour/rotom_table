# UML CASE Tool

Computer-aided software engineering (CASE) tools store [[uml-model|model]] information in a central database, allowing each diagram to display a specific [[uml-view|view]]. When a class name changes in a [[uml-class-diagram|class diagram]], related [[uml-statechart-diagram|statechart diagrams]] update automatically.

The UML meta-model provides the database structure, defining which elements are allowed and their properties.

CASE tools can be resource-intensive and expensive, but their use is worthwhile for projects beyond minimal scope.
