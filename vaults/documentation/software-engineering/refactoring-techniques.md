# Refactoring Techniques

A catalog of proven transformations for restructuring code without changing its behavior. Each technique addresses specific [[code-smells]] and moves code toward [[clean-code]].

## Categories

- [[composing-methods-techniques]] — streamlining methods by extracting, inlining, and simplifying their internals
- [[moving-features-techniques]] — redistributing responsibilities between classes
- [[organizing-data-techniques]] — improving how data is stored, accessed, and represented
- [[simplifying-conditionals-techniques]] — reducing complexity in branching logic
- [[simplifying-method-calls-techniques]] — making method interfaces cleaner and more intuitive
- [[dealing-with-generalization-techniques]] — managing inheritance hierarchies and abstractions
