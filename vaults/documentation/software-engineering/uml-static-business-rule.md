# UML Static Business Rule

A [[uml-business-rules|business rule]] that can be verified at any time and deals with class structures. Documented in [[uml-class-diagram|class diagrams]] within the [[uml-structural-view|structural view]].

Examples:
- "A ticket belongs to exactly one customer" (expressed as multiplicity in a class diagram)
- "Each seat can only be assigned to one passenger per flight"
