Sprint is a combat action that trades the standard action for increased movement. Implemented via `POST /api/encounters/:id/sprint`.

The combatant gains a Sprint temporary condition and +50% movement speed for the turn. The standard action is consumed. The [[movement-modifiers-utility]] applies the Sprint multiplier when calculating available movement.
