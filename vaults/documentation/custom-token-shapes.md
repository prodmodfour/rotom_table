Pokemon with non-standard body shapes (serpents, long-bodied creatures) can use custom grid footprints instead of the default square. The custom shape should have roughly the same total squares as the default for that size category.

A Gigantic Steelix might be 8x2 instead of 4x4, twisting into different shapes as it moves. This means [[size-determines-grid-footprint]] defines the area budget, not a fixed shape. The app's token system must support non-square footprints for creatures where a square doesn't make sense.
