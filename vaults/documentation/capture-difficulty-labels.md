# Capture Difficulty Labels

Numeric [[capture-rate-formula|capture rates]] map to human-readable labels for display.

| Threshold | Label |
|---|---|
| ≥ 80 | Very Easy |
| ≥ 60 | Easy |
| ≥ 40 | Moderate |
| ≥ 20 | Difficult |
| ≥ 1 | Very Difficult |
| < 1 | Nearly Impossible |

The UI color-codes these (green through red, grayed out for fainted/impossible).
