# Pokemon Stat Allocation

Level-up stat point allocation. Each level grants [[five-stat-points-per-level|5 stat points]], distributed freely across any stats with no ordering constraints per [[base-stat-relations-removed]].

Total budget = 5 x level. The allocation workflow allows incremental or batch stat point assignment. [[pokemon-hp-formula|HP]] is recalculated when HP stat points are allocated.
