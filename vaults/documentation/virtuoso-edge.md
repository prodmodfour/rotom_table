Virtuoso is a high-tier edge that provides an enhanced bonus to a specific skill. Like other skill-modifying edges, it is stored as a [[structured-edge-objects|structured edge object]] with metadata identifying the target skill and bonus tier.

Virtuoso stacks with or supersedes [[skill-enhancement-edge]] depending on the skill — the metadata encodes the final bonus so the app doesn't need to resolve stacking rules at runtime.
