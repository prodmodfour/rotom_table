# UML Enterprise Application Integration

Enterprise Application Integration (EAI) encompasses methods, concepts, and tools for the classification, connection, and coordination of applications within organizations. The objective is integrated business processing through networks of internal applications across different generations and architectures.

Prerequisites include unified documentation of business processes, application networks, individual applications, and their [[uml-system-integration-interface|interfaces]]. [[uml]] serves this purpose, with the OMG creating a dedicated *UML Profile for Enterprise Application Integration*.
