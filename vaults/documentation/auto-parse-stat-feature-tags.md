When a trainer feature contains machine-parseable `[+Stat]` patterns (e.g., `[+Attack]`, `[+Special Defense]`), the app auto-detects and applies the stat bonuses. This eliminates manual bookkeeping errors and ensures stat totals are always accurate.

This is [[automate-routine-bookkeeping]] applied to feature text. The app handles tracking; the GM handles decisions.
