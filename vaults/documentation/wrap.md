Wrap is stored in [[movedata-reference-table]] with `damageClass: "Static"`, `type: "Normal"`, `damageBase: null`, `frequency: "Static"`, `ac: null`, `range: "--"`.

## Frequency

Static frequency means this move cannot be used actively in combat. It provides a passive effect only. The [[move-frequency-system]] blocks all use attempts.

## Resolution

As a Status move, Wrap skips the [[damage-flow-pipeline]]. No accuracy roll is needed (AC is null).

## Ability Interactions

No ability-interaction flags are set in the [[moves-csv-source-file]].
