Skill Enhancement edges add a flat bonus to a specific skill check. These are stored as [[structured-edge-objects]] with metadata identifying the enhanced skill.

The app reads the metadata to apply the bonus automatically during skill resolution, following the [[automate-routine-bookkeeping]] principle — the trainer shouldn't have to remember to add the bonus manually each time.
