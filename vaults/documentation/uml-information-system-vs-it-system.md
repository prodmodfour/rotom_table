# Information System vs IT System

An **information system** manages information needed for business processes. Information systems predate computers — historical examples include stock lists from Knossos (3,500 years ago), Roman merchants' records, and Hanseatic trading office ledgers using chalkboards, filing systems, and index cards.

An **IT system** is a computer-based information system that provides information in response to user queries.

The key insight: an information system *can* be implemented as IT technology, but conceptually it has nothing to do with computers. It can be realized in many ways.

## Real-world vs information system representation

Objects in a business system (passengers, tickets, flights) have corresponding representations — *images* — in the information system. These images consist of stored information about the real objects.

Within business processes, information is retrieved from and stored in IT systems. The [[uml]] modeling techniques apply not only for IT system development but whenever an information system needs analysis.
