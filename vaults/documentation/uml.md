# UML

Unified Modeling Language — a standardized visual language for specifying, constructing, and documenting the artifacts of systems. Maintained by the Object Management Group (OMG).

UML provides multiple diagram types reflecting different views of a system. It applies to business system modeling, IT system modeling, and system integration modeling.

## Diagram types

- [[uml-use-case-diagram]] — actors, use cases, and their relationships
- [[uml-activity-diagram]] — procedures and business process flows
- [[uml-sequence-diagram]] — chronological message exchange between objects
- [[uml-class-diagram]] — classes, attributes, associations, and multiplicity
- [[uml-package-diagram]] — organization units and their contents
- [[uml-statechart-diagram]] — states, transitions, and events governing object behavior
- [[uml-communication-diagram]] — object interaction topology and query flow
