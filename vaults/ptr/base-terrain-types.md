PTU defines three foundational terrain types that determine what can traverse them. Regular Terrain (dirt, short grass, cement, smooth rock, indoor) is freely walkable. Earth Terrain is underground with no existing tunnel — only Burrow-capable Pokemon can shift through it. Underwater Terrain is submerged water — only Swim-capable Pokemon can move through it.

These base types exist alongside the modifier terrains ([[slow-terrain-doubles-movement]], [[rough-terrain-intervening-only]], [[blocking-terrain-impassable]]) in the [[multi-tag-terrain-system]]. The base type determines traversal permission; modifiers alter movement cost or accuracy.
