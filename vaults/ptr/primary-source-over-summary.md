When an ability description and a summary table disagree on a mechanical value, the ability description takes precedence. Ability descriptions are the primary source of truth; summary tables are secondary quick-references that may contain transcription errors.

This is why [[sun-blanket-heals-a-tick]] uses the Tick value (1/10th max HP) from the ability description rather than the 1/16th from the weather summary table.
