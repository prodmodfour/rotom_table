Teleport is a Psychic-type [[type-effectiveness-excludes-status-moves|Status]] move with no Damage Base, [[scene-frequency-definition|Scene]] frequency, no AC, and a range of Self, Interrupt.

The user Teleports up to X meters, where X is its Teleporter Capability. Any Move that targeted Teleport's user continue through the desired target's space if the Move allows for it as if the user hadn't been there; single target moves simply miss.

Teleport grants the Teleporter 4 [[move-granted-capabilities|capability]] while known.
