Spit Up is a Normal-type [[type-effectiveness-excludes-status-moves|Special]] move with a Damage Base of X, See Effect, [[scene-frequency-definition|Scene]] frequency, AC 2, and a range of 4, 1 Target.

For each Stockpiled Count the user has, Spit Up’s Damage Base is increased by +8. If the user has no Stockpiled count, Spit Up cannot be used.
