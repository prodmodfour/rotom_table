Endure is a Normal-type [[type-effectiveness-excludes-status-moves|Status]] move with no Damage Base, [[scene-frequency-definition|Daily]] frequency, no AC, and a range of Self, Reaction, Trigger.

If the user is hit by a damaging Move, you may use Endure as a Free Action. If the Move would bring Endure's user down to 0 HP or less, Endure's user instead is set to 1 HP.

As a Daily move, Endure can be used once per day, refreshing on [[extended-rest-refreshes-daily-moves|extended rest]]. It is still limited to [[daily-moves-once-per-scene|one use per scene]].
