# Moves (Base PTU)

~815 move stat blocks from PTU 1.05. Each file records: type, damage class, damage base, accuracy, range, frequency, and effect text.

- When a move has no file in `../ptr_move_redesigns/`, these are the canonical stats.
- When a PTR redesign exists, compare both to understand what changed and why.
- Moves use `[[wikilinks]]` to reference rules like `[[stab-adds-to-damage-base]]` and `[[melee-range-is-adjacency]]`.

## Starting nodes

- `hidden-power.md` — only move whose type varies per individual (d20 roll)
- `struggle.md` — fallback attack (see also `../rules/struggle-is-not-a-move.md`)
