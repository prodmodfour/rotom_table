PTR removes skill ranks and the d6 dice pool — see [[skill-ranks-removed]] and [[skill-check-1d20-plus-modifier]].

In PTU, each skill rank mapped to a fixed number of d6s: Pathetic = 1d6, Untrained = 2d6, Novice = 3d6, Adept = 4d6, Expert = 5d6, Master = 6d6. This linear progression was the foundation of the [[skill-check-xd6-vs-dc]] mechanic.

The app computes dice pools automatically from stored rank values — this is [[automate-routine-bookkeeping]] applied to skill checks.
