All Persistent conditions in PTR now include a save check, either at the end or beginning of the affected target's turn. This is a blanket rule — no Persistent condition is exempt.

Previously, some Persistent conditions (like [[frozen-cured-by-damage-types|Frozen]]) had save checks while others (like Burn, Poison) did not. PTR standardizes this: every Persistent condition gives the afflicted a chance to shake it off each turn.

Specific save DCs and timing vary per condition — see [[burning-save-dc-18]], [[poisoned-save-dc-10]], [[badly-poisoned-save-dc-15]].
