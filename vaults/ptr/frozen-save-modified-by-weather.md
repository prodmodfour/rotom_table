The Frozen status save check (DC 16) receives a +4 bonus in [[sunny-weather-effects|Sunny Weather]] and a -2 penalty in [[hail-weather-effects|Hail]]. This makes weather a factor in status condition management — Sunny Weather helps thaw frozen allies, while Hail makes Frozen more oppressive.

This is a key weather-status interaction alongside [[frozen-cured-by-damage-types]], which handles the damage-based cure path.
