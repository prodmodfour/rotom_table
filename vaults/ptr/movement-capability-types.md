In PTR, movement uses [[movement-traits]]: Land, Sky, Water, Phase, Burrow, and Teleporter. Each is a [[trait-definition|trait]] that grants X movement in that mode per turn.

This replaces PTU's capabilities (Overland, Swim, Sky, Burrow, Teleporter).
