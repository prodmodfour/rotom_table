# Telekinetic [X] [Y]

## Properties
	- Category: Innate
	- Activation: Active (Swift Action)
	- Energy Cost: x * y (where x is the weight class being lifted, up to X, and y is the squares moved, up to Y)

## Effect

The user can telekinetically lift and manipulate objects up to Weight Class X and move them up to Y squares per turn. The energy cost scales with how much of this capacity is used — lifting a lighter object a short distance costs less than lifting at full capacity at full speed.

The user can interact with objects at range as though it had hands, including operating simple mechanisms (doors, latches, levers) and picking up or setting down items.

Telekinetic manipulation requires line of sight to the object. If line of sight is broken, the object is dropped.

The user cannot wield weapons, use items that require fine motor control (e.g. writing, lockpicking), or make attacks with held objects unless another trait or move explicitly allows it.

## Styling

The correct format is `Telekinetic [X] [Y]` where X and Y are numbers. Each parameter gets its own brackets. Do not omit the brackets (e.g. `Telekinetic 4` or `Telekinetic 2 3` is incorrect).

## Notes

This trait does not grant any combat benefit on its own. It represents ambient supernatural manipulation of the environment — poltergeist activity, not psychic combat.
