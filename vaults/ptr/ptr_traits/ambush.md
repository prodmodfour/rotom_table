# Ambush

## Properties
	- Category: Innate
	- Activation: Triggered (Voluntary)
	- Energy Cost: 0

## Effect

If the user attacks a target from Stealth or from a concealed position (e.g. burrow, cave hole), the first attack gains +2 to the Accuracy Check.
