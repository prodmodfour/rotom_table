# Hive Seeker [X]

## Properties
	- Category: Innate
	- Activation: Passive
	- Energy Cost:

## Effect

The user is compelled to find and serve a [[Vespiquen]]. If the user does not currently belong to a hive or serve a queen, it will actively search for one — wandering away from camp, investigating pheromone trails, and approaching any Vespiquen it encounters with submissive behavior. The user will attempt to leave its trainer's side to join a wild hive if one is nearby.

If the user already serves a Vespiquen (including a trainer's Vespiquen), this instinct is satisfied and does not trigger.

This can be suppressed for 1 hour with a DC 10 + X Command, Charm, Persuasion or Intimidate check, modified by the Pokemon's [[pokemon-social-skill-hierarchy|social skill hierarchy]]. If you fail this check, you cannot try again for 1 hour.

## Notes

[[Combee]] are eusocial insects — a Combee without a colony is biologically incomplete. This trait represents the deep drive to find purpose within a hive hierarchy. Trainers who keep Combee without a Vespiquen must regularly suppress this instinct or risk their Combee wandering off to join a wild hive.
