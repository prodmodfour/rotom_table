# Shell [X]

## Properties
	- Category: Innate
	- Activation: Passive
	- Energy Cost: 0

## Effect

The user has a hard outer covering — a shell, exoskeleton, carapace, or similar protective structure. Damage dealt to the user is reduced by X (minimum 0).

## Unlock Conditions


## Styling

The correct format is `Shell [X]` where X is a number. Do not omit the brackets (e.g. `Shell 2` is incorrect).

## Notes

X ranges from 1 to 5. A higher value represents a thicker, tougher, or more encompassing shell.
