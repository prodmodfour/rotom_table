# Run Away

## Properties
	- Category: Innate
	- Activation: Passive
	- Energy Cost: —

## Effect

The user's movement does not provoke [[attack-of-opportunity-trigger-list|Attacks of Opportunity]].

## Unlock Conditions


## Notes

Reflects a prey animal's instinct to bolt at the first sign of danger — nimble enough to slip away before a predator can react. In PTU, this was the Run Away Ability (guaranteed escape from wild battles). In PTR, [[disengage-avoids-opportunity-attacks|Disengage]] is available to everyone; Run Away simply removes the need for it.
