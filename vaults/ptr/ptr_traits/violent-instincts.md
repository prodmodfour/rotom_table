# Violent Instincts [X]

## Properties
	- Category: Innate
	- Activation: Passive
	- Energy Cost:

## Effect

When the user encounters a creature toward which its disposition is not Neutral or better, the user will immediately attack that creature — unless the user is already engaged in combat with a target.

This can be suppressed for 1 hour with a DC 10 + X Command, Charm, Persuasion or Intimidate check, modified by the Pokemon's [[pokemon-social-skill-hierarchy|social skill hierarchy]]. If you fail this check, you cannot try again for 1 hour.

## Styling

The correct format is `Violent Instincts [X]` where X is a number. Do not omit the brackets (e.g. `Violent Instincts 2` is incorrect).
