# Phaser [X]

## Properties
	- Category: Innate
	- Activation: Passive

## Effect

Grants [[movement-traits|Phase X movement]]. Phase movement allows the user to pass through solid objects and terrain as though they were not there, ignoring [[slow-terrain-doubles-movement|Slow Terrain]] and other physical obstacles.

## Styling

The correct format is `Phaser [X]` where X is a number. Do not omit the brackets (e.g. `Phaser 2` is incorrect).
