# Landwalker X

## Properties
	- Category: Innate
	- Activation: Passive
	- Energy Cost:

## Effect

The user can move X squares per turn on land. This is the standard ground movement mode for most Pokemon.

## Styling

The correct format is `Landwalker X` where X is a number (e.g. `Landwalker 4`).

## Notes

Adapted from PTU's Overland capability. Nearly every Pokemon has Landwalker — it is the default locomotion mode. Values typically range from 2 (slow/aquatic) to 8 (fast runners). The typical value is 4.
