# Stance Change

## Properties
	- Category: Innate
	- Activation: Triggered (Involuntary) / Active (Swift Action)
	- Energy Cost: 0

## Effect

The user has two stances: Shield Forme (default) and Blade Forme. Changing stance swaps the user's Attack and Defense individual stats and their Special Attack and Special Defense individual stats, without changing Combat Stages.

Stance changes occur automatically:
- Using a damaging attack switches to Blade Forme.
- Using [[kings-shield|King's Shield]], Protect, or a Status Move that raises Defense Combat Stages switches to Shield Forme.

The user may also change stance voluntarily as a Swift Action.

While wielded as a [[living-weapon|Living Weapon]], the user is locked in Blade Forme.

## Unlock Conditions

- Species: [[Aegislash]]

## Notes

The tension is that the user cannot be both offensive and defensive at the same time. Shield Forme is the default -- Aegislash enters combat as an impenetrable wall. The moment it commits to an attack, it exposes itself. The only way back to Shield Forme mid-combat is through protective moves or spending a Swift Action without attacking. This creates a natural rhythm: wall up, wait for an opening, strike, then scramble to defend again.

The stat swap is on [[individual-stats-vs-base-stats|individual stats]], not [[base-stats-unmodifiable-by-traits|base stats]]. Combat Stages are unaffected -- a +2 Attack CS in Shield Forme is still +2 Attack CS in Blade Forme, but the underlying stat it multiplies changes.
