# Hydration

## Properties
	- Category: Innate
	- Activation: Passive
	- Energy Cost: 0

## Effect

While [[rain-weather-effects|Rain]] is active, the user cures one status condition at the end of each of its turns.

## Unlock Conditions


## Notes

The user chooses which status condition to cure if affected by more than one. This includes persistent conditions like Burn, Paralysis, Poison, Freeze, and Sleep. It does not cure injuries or remove Combat Stages.
