# Barrier [X]

## Properties
	- Category: Innate
	- Activation: Passive
	- Energy Cost: 0

## Effect

The user can project barriers of energy, force, or hardened material. X is the barrier's strength rating.

## Unlock Conditions


## Styling

The correct format is `Barrier [X]` where X is a number. Do not omit the brackets (e.g. `Barrier 1` is incorrect).
