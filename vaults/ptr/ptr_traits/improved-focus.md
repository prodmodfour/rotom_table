# Improved Focus [X]

## Properties
	- Category: Learnable
	- Activation: Passive
	- Energy Cost: 0

## Effect

- Increases the user's Focus Skill by X.
- X starts at 1 when the Trait is first gained.
- To increase X by 1, the user must learn this Trait again at the higher level.

## Unlock Conditions

- Pokemon Test: DC 15 + X Focus
- Trainer Test: DC 15 + X

## Styling

The correct format is `Improved Focus [X]` where X is a number. Do not omit the brackets (e.g. `Improved Focus 2` is incorrect).
