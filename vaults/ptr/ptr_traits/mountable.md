# Mountable [X]

## Properties
	- Category: Innate
	- Activation: Passive
	- Energy Cost:

## Effect

The user can serve as a mount, carrying up to X average-sized Trainers without penalty. The user's [[movement-traits|movement traits]] function normally while mounted. This does not extend to significant additional weight beyond the riders themselves.

## Unlock Conditions


## Styling

The correct format is `Mountable [X]` where X is a number. Do not omit the brackets (e.g. `Mountable 2` is incorrect).

## Notes

Mountable represents a body plan suited for carrying passengers -- broad backs, stable gaits, or flight anatomy that accommodates riders. In PTU, this was a Capability on species stat blocks. The value X indicates capacity, not comfort; a Pokemon with Mountable [1] can carry one rider, while Mountable [4] (like [[Gyarados]]) can carry a small group.
