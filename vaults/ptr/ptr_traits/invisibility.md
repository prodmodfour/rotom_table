# Invisibility

## Properties
	- Category: Innate
	- Activation: Active (Shift Action)

## Effect

As a Shift Action, the user turns invisible. While invisible, attacks against the user suffer +4 to their Accuracy Check. The user cannot perform Moves while invisible.

The user can remain invisible for up to 4 minutes, and can become visible again as a Free Action. After becoming visible, the user must wait 2 minutes plus the number of minutes spent invisible before turning invisible again.
