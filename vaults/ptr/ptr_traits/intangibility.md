# Intangibility

## Properties
	- Category: Innate
	- Activation: Active (Standard Action)

## Effect

As a Standard Action, the user turns completely Intangible. While Intangible, the user cannot be targeted by Moves or attacks, cannot perform Standard Actions, and loses a Tick of HP at the end of each round.
