# Combee Command

## Properties
	- Category: Innate
	- Activation: Passive
	- Energy Cost:

## Effect

The user can command a swarm of [[Combee]] through pheromone signals. The user controls up to 10 Combee per point of Command. Combee under this control must be at or below one quarter of the user's level. The user can communicate telepathically with Combee that serve her within a 3 mile radius.

## Unlock Conditions


## Notes

This trait represents the [[Vespiquen]]'s role as colony queen. Her pheromone output determines colony size — a stronger Vespiquen commands a larger hive. The level restriction reflects that a queen's chemical authority only extends to workers significantly less developed than herself.
