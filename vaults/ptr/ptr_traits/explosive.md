# Explosive

## Properties
	- Category: Innate
	- Activation: Triggered (Involuntary)
	- Energy Cost: 0

## Effect

When the user faints, it releases a burst of volatile energy. This creates a DB 6 Normal Special Attack targeting all adjacent creatures (Burst 1 centered on the user). This attack uses the user's Special Attack stat and cannot be prevented or suppressed. The user does not need to be conscious to trigger this effect.
