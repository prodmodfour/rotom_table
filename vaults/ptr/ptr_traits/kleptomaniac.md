# Kleptomaniac [X]

## Properties
	- Category: Innate
	- Activation: Passive
	- Energy Cost:

## Effect

When the user sees an unattended item or a held item on a creature toward which its disposition is not Friendly or better, the user will attempt to steal that item at the first opportunity.

This can be suppressed for 1 hour with a DC 10 + X Command, Charm, Persuasion or Intimidate check, modified by the Pokemon's [[pokemon-social-skill-hierarchy|social skill hierarchy]]. If you fail this check, you cannot try again for 1 hour.

## Styling

The correct format is `Kleptomaniac [X]` where X is a number. Do not omit the brackets (e.g. `Kleptomaniac 1` is incorrect).
