# Swimmer X

## Properties
	- Category: Innate
	- Activation: Passive
	- Energy Cost:

## Effect

The user can move X squares per turn through water. The user can breathe and act normally while submerged.

## Styling

The correct format is `Swimmer X` where X is a number (e.g. `Swimmer 4`).

## Notes

Adapted from PTU's Swim capability. Most Pokemon have at least Swimmer 2 — a baseline ability to paddle or float. Higher values represent species adapted for aquatic life. Values typically range from 2 (can swim) to 8 (fast aquatic predator).
