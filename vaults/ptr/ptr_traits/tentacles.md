# Tentacles [X]

## Properties
	- Category: Innate, Natural Weapon, Bludgeoning
	- Activation: Passive
	- Energy Cost:

## Effect

The user has flexible tentacles that can be used to grasp, lash, and strike in battle.

The user can manipulate objects and interact with the environment using its tentacles as though it had hands, at a reach of X squares. The user can attempt grapple checks using its tentacles with a +X bonus.

## Unlock Conditions


## Styling

The correct format is `Tentacles [X]` where X is a number. Do not omit the brackets (e.g. `Tentacles 2` is incorrect).
