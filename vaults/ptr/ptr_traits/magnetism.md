# Magnetism [X]

## Properties
	- Category: Innate
	- Activation: Active (Swift Action)
	- Energy Cost: w * d (where w is the weight class being lifted, up to X, and d is the squares moved, up to X+2)

## Effect

The user can magnetically lift and manipulate metallic or ferromagnetic objects up to Weight Class X and move them up to X+2 squares per turn. The energy cost scales with how much of this capacity is used — lifting a lighter object a short distance costs less than lifting at full capacity at full range.

The user can interact with metallic objects at range, including operating metallic mechanisms (latches, locks, levers) and picking up or setting down metallic items.

Magnetic manipulation requires line of sight to the object. If line of sight is broken, the object is dropped.

The user cannot wield weapons, use items that require fine motor control (e.g. lockpicking), or make attacks with held objects unless another trait or move explicitly allows it.

## Styling

The correct format is `Magnetism [X]` where X is a number. Do not omit the brackets (e.g. `Magnetism 2` is incorrect).

## Notes

This trait is a limited form of [[telekinetic]] — it functions identically but only affects metallic or ferromagnetic objects. Non-metallic objects cannot be manipulated.

Magnetism represents ambient electromagnetic manipulation of the environment — an extension of electrokinetic ability into sustained object control.
