PTR removes the Features, Edges, and Classes systems from PTU. These applied to both trainers and Pokemon in PTU. In PTR, character customization is handled entirely through [[trait-definition|Traits]].

This is part of the same simplification that [[traits-replace-abilities-capabilities-natures|replaces Abilities, Capabilities, and Natures with Traits]]. For skills, this means Features and Edges no longer serve as prerequisites — see [[skill-modifiers-from-traits-or-circumstance]].
