# Move Descriptions

~378 narrative flavor texts describing what each move looks like in action. Each links to its stat block in `../ptr_moves/`.

## What you can't know without exploring here

- How a specific move looks and feels narratively (animation, sound, visual effects)
- Which moves have flavor text vs which are still undescribed

- These feed the app's move detail UI panels.
- One description per move — not all moves have descriptions yet.
- Descriptions are pure flavor, not mechanical. They don't override stat block data.
