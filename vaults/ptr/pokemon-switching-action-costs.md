Pokemon switching has variable action costs depending on the situation:

- **Full Switch** (recall + release): Standard Action. Requires the Pokemon to be within 8m recall range.
- **Fainted Pokemon switch:** Shift Action (cheaper because fainted Pokemon are dead weight).
- **Recall only or Release only:** Each is a Shift Action individually.
- **Dual recall or dual release:** One Standard Action for two Pokemon simultaneously.

Recall and Release as separate Shift Actions in the same round still counts as a Switch. You cannot recall and release the same Pokemon in one round.
