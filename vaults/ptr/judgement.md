Judgement is a Normal-type [[type-effectiveness-excludes-status-moves|Special]] move with a Damage Base of 10, [[scene-frequency-definition|Daily]] frequency, AC 2, and a range of 6, Ranged Blast 3, Smite.

Judgment's Type can be whatever Elemental Type the user wants it to be.

As a Daily move, Judgement can be used once per day, refreshing on [[extended-rest-refreshes-daily-moves|extended rest]]. It is still limited to [[daily-moves-once-per-scene|one use per scene]].
