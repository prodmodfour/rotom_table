Character creation follows a nine-step workflow:

1. Create Character Concept
2. Create Skill Background — see [[background-shapes-starting-skills]]
3. Choose Edges — see [[starting-edge-budget]]
4. Choose Features — see [[starting-feature-budget]]
5. Assign Combat Stats — see [[starting-stat-allocation]]
6. Find Derived Stats — see [[trainer-hp-formula]], [[evasion-from-defensive-stats]], [[traits-derived-from-skills]]
7. Create Basic Descriptions
8. Choose Starter Pokémon
9. Buy Starting Items

Steps 3 and 4 can be taken in any order per [[edges-and-features-interleave]].
