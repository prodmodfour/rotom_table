# Endeavor

## Properties
	- Type: Normal
	- Damage Class: Physical
	- Damage Base: —
	- AC: 2
	- Range: Melee, 1 Target, Dash
	- Energy Cost: 5

## Effect
The target loses a Tick of Hit Points for each Injury the user has.

## Unlock Conditions
Level 25

## Notes
Damage scales with user's injuries, becoming more dangerous the more hurt the user is. Dash keyword allows closing distance. Contest Type: Tough. Contest Effect: Double Time.
