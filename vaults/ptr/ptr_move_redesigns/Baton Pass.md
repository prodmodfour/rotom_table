# Baton Pass

## Properties
	- Type: Normal
	- Damage Class: Status
	- Damage Base: —
	- AC: —
	- Range: Self
	- Energy Cost: 0

## Effect
The user is replaced with another Pokemon from their trainer's roster. All Combat Stage changes, Coats, and Stratagems on the user are transferred to the replacement. Baton Pass may be used to switch even if the user is Trapped.

## Unlock Conditions
Level 15

## Notes
Unique switching move that transfers stat boosts. Bypasses Trap effects. Contest Type: Cute.
