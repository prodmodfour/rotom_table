# Spit Up

## Properties
	- Type: Normal
	- Damage Class: Special
	- Damage Base: X
	- AC: 2
	- Range: 4, 1 Target
	- Energy Cost: 5

## Effect
For each Stockpiled Count the user has, Spit Up's Damage Base is increased by +8. If the user has no Stockpiled count, Spit Up cannot be used.

## Unlock Conditions
Level 15

## Notes
Requires Stockpile. Damage scales with Stockpile count (max DB 24 at 3 Stockpiles). Contest Type: Tough. Desperation.
