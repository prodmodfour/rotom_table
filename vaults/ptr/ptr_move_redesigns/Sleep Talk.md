# Sleep Talk

## Properties
	- Type: Normal
	- Damage Class: Status
	- Damage Base: —
	- AC: —
	- Range: Self
	- Energy Cost: 3

## Effect
Select another of the user's Moves at random; this turn, the user may Shift and use that Move despite being Asleep. Sleep Talk can only be used by Sleeping targets.

## Unlock Conditions
Level 15 or TM

## Notes
Can only be used while Asleep. Contest Type: Cute. Steady Performance.
