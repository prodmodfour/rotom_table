# Haze

## Properties
	- Type: Ice
	- Damage Class: Status
	- Damage Base: —
	- AC: —
	- Range: Field

## Effect
The Combat Stages of the user and all Pokemon and Trainers in the encounter are set to their default state (usually 0).

## Unlock Conditions
- Ice Manipulation 3
- Wind Manipulation 3
