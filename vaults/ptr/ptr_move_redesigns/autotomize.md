# Autotomize

## Properties
	- Type: Steel
	- Damage Class: Status
	- Damage Base: —
	- AC: —
	- Range: Self
	- Energy Cost: 1

## Effect
For the remainder of the Encounter, the user's Weight Class is one value lower, to a minimum of 1. If the user can, the user's Speed is raised by +2 Combat Stages.

## Unlock Conditions
— Steel Type
- Technology Education 3

## Notes
Speed boost plus weight reduction. Especially useful for heavy Steel-types.
