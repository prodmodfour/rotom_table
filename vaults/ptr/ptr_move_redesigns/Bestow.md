# Bestow

## Properties
	- Type: Normal
	- Damage Class: Status
	- Damage Base: —
	- AC: —
	- Range: Melee, 1 Target
	- Energy Cost: 0

## Effect
The user gives its held item to the target, unless the target is already holding an item. Using Bestow is a Swift Action.

## Unlock Conditions
Level 5

## Notes
Swift Action item transfer. Niche utility move. Contest Type: Cute.
