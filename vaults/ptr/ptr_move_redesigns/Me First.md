# Me First

## Properties
	- Type: Normal
	- Damage Class: Status
	- Damage Base: —
	- AC: —
	- Range: Self, Trigger, Interrupt
	- Energy Cost: 3

## Effect
If an opponent declares a Damaging Attack against the user, and Me First's user has a higher Speed stat than the target, the user may use Me First as an Interrupt. The user will then use the same Move the triggering foe was about to use on that foe.

## Unlock Conditions
Level 20

## Notes
Copying move. Requires higher Speed to activate. Contest Type: Cute. Contest Effect: Saving Grace.
