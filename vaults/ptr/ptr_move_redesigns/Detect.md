# Detect

## Properties
	- Type: Fighting
	- Damage Class: Status
	- Damage Base: —
	- AC: —
	- Range: Self, Interrupt, Shield, Trigger
	- Energy Cost: 3

## Effect
If the user is hit by a Move, the user may use Detect. The user is instead not hit by the Move. The user takes no damage and is not affected by any of the Move's effects.

## Unlock Conditions
Level 20

## Notes
Complete evasion move usable as an Interrupt. Functions identically to Protect. Contest Type: Cool. Contest Effect: Inversed Appeal.
