# Endure

## Properties
	- Type: Normal
	- Damage Class: Status
	- Damage Base: —
	- AC: —
	- Range: Self, Reaction, Trigger
	- Energy Cost: 4

## Effect
If the user is hit by a damaging Move, the user may use Endure as a Free Action. If the Move would bring the user down to 0 Hit Points or less, the user is instead set to 1 Hit Point.

## Unlock Conditions
Level 20

## Notes
Survival move preventing KO from a single attack. Daily frequency makes it a one-time safety net. Contest Type: Tough. Contest Effect: Sabotage.
