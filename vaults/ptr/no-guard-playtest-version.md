No Guard uses the 2016 playtest packet version: +3 bonus to all attack rolls for the user, +3 bonus to all attack rolls against the user, affecting all attack types (melee and ranged).

The core rulebook version (evasion-based, melee-only) was overridden because the playtest version is simpler, already implemented, and represents more recent design intent. This adoption required a [[per-conflict-decree-required]] — the playtest packet is not errata and does not automatically override core text.
