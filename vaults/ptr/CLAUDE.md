# PTR Vault

~1760 atomic notes describing the PTR game system (a fork of PTU 1.05). Notes link to each other with `[[wikilinks]]`. Obsidian resolves links by filename regardless of folder.

## Subfolders

- `rules/` — ~376 core game mechanics (combat, stats, conditions, energy, capture, training). Start here for system rules. Has its own CLAUDE.md with starting nodes.
- `moves/` — ~815 base PTU move stat blocks. Canonical when no PTR redesign exists. Has its own CLAUDE.md.
- `move_descriptions/` — ~378 narrative flavor texts for how moves look/feel. Has its own CLAUDE.md.
- `pokemon_ecology/` — ~190 species natural history, ecology, evolution condition design. Has its own CLAUDE.md.
- `ptr_move_redesigns/` — ~382 PTR-modified move stat blocks (energy costs, no frequencies). Has its own CLAUDE.md.
- `ptr_pokemon_redesigns/` — ~97 Pokemon species redesigns (stats, types, traits, move lists). Has its own CLAUDE.md.
- `ptr_traits/` — ~174 custom trait definitions. Has its own CLAUDE.md.

## Routing

- Looking up a **move's stats**? Check `ptr_move_redesigns/` first, fall back to `moves/`.
- Looking up a **move's flavor text**? Check `move_descriptions/`.
- Looking up a **Pokemon species**? Check `ptr_pokemon_redesigns/` for stats, `pokemon_ecology/` for lore.
- Looking up a **trait**? Check `ptr_traits/`.
- Looking up a **game rule or mechanic**? Check `rules/`.
