Moves that can be used multiple times Daily can still only be used once per Scene. A "Daily 2" move can be used in two different scenes during the day, but never twice in the same scene. This prevents daily-frequency moves from overwhelming a single encounter.

This cross-frequency constraint connects [[scene-frequency-definition]] with [[extended-rest-refreshes-daily-moves]] — the daily budget is the outer limit, the scene is the inner pacing mechanism.
