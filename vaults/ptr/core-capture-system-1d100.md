The app uses the core 1d100 capture system (PTU p.214) exclusively. The errata d20 capture system is labeled "Sept 2015 Playtest" and does not represent a canonical replacement per [[errata-corrections-not-replacements]].

The 1d100 system provides more granular capture rates and is the established standard in PTU 1.05.
