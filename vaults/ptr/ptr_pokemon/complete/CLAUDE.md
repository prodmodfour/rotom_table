# Complete Pokemon Species

77 finalized species. These have been reviewed and are considered done.

## What you can't know without exploring here

- The final PTR stats, types, and trait lists for these species
- How traits progress across finished evolutionary lines

## What you'd learn

Each file is a single species following the template in `../template.md`. Files are named by species (lowercase, e.g. `aegislash.md`, `alolan grimer.md`).

## When to modify

These are finished — only edit to fix errors or apply system-wide rule changes. For new work, use `../incomplete/`.
