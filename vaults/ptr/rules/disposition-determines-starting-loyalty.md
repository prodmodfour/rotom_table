A wild Pokemon's disposition at the moment of capture determines its starting [[loyalty-rank-names|loyalty rank]], not a flat default.

| Disposition at Capture | Starting Loyalty | Rationale |
|---|---|---|
| Very Friendly | 3 (Fond) | Wanted to come with you |
| Friendly | 3 (Fond) | Willing partnership |
| Neutral | 2 (Wary) | Standard wild catch per RAW |
| Fearful | 2 (Wary) | Scared but not resentful |
| Hostile | 1 (Distrustful) | Resisted, feels undeservedly caught |
| Very Hostile | 0 (Defiant) | Deeply antagonistic, active defiance |

This refines [[loyalty-varies-by-origin]] — the origin `wild` default of Loyalty 2 applies only to Neutral disposition. Friendlier catches start higher; hostile ones start lower.
