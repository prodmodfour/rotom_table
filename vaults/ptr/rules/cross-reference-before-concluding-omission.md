Before concluding that a rule or mechanic is "deliberately omitted," cross-reference multiple chapters. An item missing from one reference table but explicitly named in shop stock lists, class features, or other chapters is an editing error, not a design choice.

This is why [[awakening-exists-as-standard-cure]] — it's absent from the ch9 gear table but named in the ch11 Pharmacy stock list and the ch4 Apothecary class. Two independent references override one table omission.
