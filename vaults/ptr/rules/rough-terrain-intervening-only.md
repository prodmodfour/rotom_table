Only rough terrain BETWEEN the attacker and target (intervening) triggers the -2 accuracy penalty. Endpoint cells (attacker's position, target's position) are excluded from the check.

PTU's Stealth capability text (p.450-451) is the strongest evidence: it writes "through Rough Terrain **or if** the Pokemon is on Rough Terrain" — using separate clauses for "through" and "on." This demonstrates PTU considers them distinct conditions. The rough terrain accuracy rule uses "through," meaning intervening terrain only.
