Type effectiveness multipliers only apply to Physical and Special moves, never Status moves. A Ghost-type Thunder Wave still paralyzes a Normal-type — the type chart doesn't block it.

The exception is type immunity to specific status conditions ([[type-grants-status-immunity]]), which operates independently of the type effectiveness system. The app must check move damage class before applying type effectiveness in [[damage-formula-step-order]].
