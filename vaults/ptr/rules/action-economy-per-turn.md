Each combatant's turn provides: one Standard Action, one [[shift-renamed-to-movement-action|Movement Action]] (previously called Shift Action), one Swift Action, and unlimited Free Actions (each triggered free action activates only once per trigger).

Using a Move, Struggle Attack, using items, throwing Poke Balls, drawing weapons, using the Pokedex, and Combat Maneuvers are all Standard Actions. Movement uses the Movement Action, which [[movement-splitting|can be split]] between other actions. The app's turn tracker should enforce this budget per [[action-costs-are-literal]].
