Size Information describes a character's physical dimensions:

- Height
- Weight
- Size Class

The size classes are: Miniscule, Tiny, Small, Medium, Large, Huge, Gigantic.

Both Pokemon and [[trainers-are-human-species|trainers]] have Size Information. For Pokemon, it is something their [[species-determines-vs-informs|species informs]]. Size Class determines [[size-determines-grid-footprint|grid footprint]].
