Encounter tables should include a disposition probability distribution for each species — a percentage chance for each of the [[wild-pokemon-six-dispositions]], totalling 100%. Each percentage should have a behavioral explanation grounded in the species' temperament.

Example — Ursaring:
- Very Friendly: 0%
- Friendly: 0%
- Neutral: 40% (usually ignores trainers)
- Fearful: 5% (rarely scared)
- Hostile: 25% (if protecting cubs)
- Very Hostile: 30% (territorial)

These weights reflect [[wild-encounter-motivations]] and [[sensible-ecosystems]] — a species' typical behavior determines how likely each disposition is when encountered.
