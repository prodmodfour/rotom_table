X-Items boost a Pokemon's Combat Stages by +2 for the duration of an encounter. They only work on Pokemon, not Trainers. Applied per [[applying-items-action-economy]].

| Item | Effect | Cost |
|---|---|---|
| X Attack | +2 Attack CS | $350 |
| X Defend | +2 Defense CS | $350 |
| X Special | +2 Special Attack CS | $350 |
| X Sp. Def | +2 Special Defense CS | $350 |
| X Speed | +2 Speed CS | $350 |
| Dire Hit | +2 Critical Hit Range | $600 |
| X Accuracy | +2 Accuracy | $600 |
| Guard Spec | Prevents CS/Accuracy reduction for 5 turns | $700 |
