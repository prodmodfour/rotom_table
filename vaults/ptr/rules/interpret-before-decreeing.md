When PTU is silent, first ask whether PTU has spoken on the surrounding topic. If PTU defines a mechanic and omits something from it, that omission is intentional — [[silence-means-no-effect]]. If PTU never addresses the topic at all, or addresses it but the result is ambiguous or broken, the silence is a gap that requires a ruling — [[per-conflict-decree-required]].

This makes interpretation and governance sequential, not competing. You exhaust interpretation first. Only when interpretation cannot yield a clear answer — because the topic is unaddressed, or because the addressed result is degenerate — do you escalate to a decree.

The test is whether PTU has "spoken." A defined mechanic that doesn't mention X means X doesn't apply. An undefined mechanic means there is nothing to interpret, and a decree must fill the gap.
