A defender applies exactly one of their three evasion values to any given accuracy check, constrained by attack class. Physical Evasion can only modify attacks targeting Defense. Special Evasion can only modify attacks targeting Special Defense. Speed Evasion may apply to any attack with an accuracy check.

The defender chooses which applicable evasion to use. This means Speed Evasion is always an option, making it the universal defensive stat. The app's accuracy resolution must enforce both the type constraint and the single-evasion-per-check rule.
