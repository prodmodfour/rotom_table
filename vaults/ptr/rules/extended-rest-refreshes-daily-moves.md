Daily-frequency Moves are regained during an Extended Rest, but only if the Move hasn't been used since the previous day. Using a Daily move and then taking an Extended Rest on the same day does not restore it — the refresh is gated by a new-day boundary, not merely by resting.

This follows [[separate-mechanics-stay-separate]]: the Extended Rest is the trigger, but the day boundary is the real constraint.
