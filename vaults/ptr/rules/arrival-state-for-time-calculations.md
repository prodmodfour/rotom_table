Time-based calculations for healing use the state at the start of the healing process, not the result. The duration reflects the effort required to treat the conditions present at arrival.

This is why [[pokemon-center-pre-healing-time]] uses the pre-healing injury count — a Pokemon arriving with 4 injuries takes 3 hours regardless of how many are healed, because the healing process must address each injury.
