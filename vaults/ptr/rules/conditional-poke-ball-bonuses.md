Many Poke Ball types have situational conditions that unlock stronger modifiers. When the condition is not met, the ball functions as a Basic Ball (+0). Each rewards a different tactical approach:

- **Level Ball:** −20 if target is under half the user's active pokemon's level
- **Love Ball:** −30 if active pokemon is same evolutionary line and opposite gender
- **Net Ball:** −20 if target is Water or Bug type
- **Dusk Ball:** −20 in dark or low-light conditions
- **Nest Ball:** −20 if target is under level 10
- **Repeat Ball:** −20 if trainer already owns the species
- **Dive Ball:** −20 if found underwater or underground
- **Fast Ball:** −20 if target has Movement above 7
- **Moon Ball:** −20 if target evolves with an Evolution Stone
- **Lure Ball:** −20 if target was baited
- **Timer Ball:** starts at +5, improves by −5 each round to max −20
- **Quick Ball:** starts at −20, decays to 0 by round 4
- **Heavy Ball:** −5 per Weight Class above 1

Ball selection becomes a strategic decision tied to encounter context.
