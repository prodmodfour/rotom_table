When an ecology note in [[pokemon_ecology|pokemon ecology]] references specific traits, those traits must actually exist in the Pokemon's species. Ecology notes and species must stay in sync -- if a trait is removed from a species, any ecology note mentioning it must be updated. If an ecology note names a trait that was never implemented, the note is aspirational and misleading.

[[lore-does-not-require-traits|Not every lore detail needs a trait]], but when a note explicitly says "the X trait reflects this," X must exist. The inverse is less strict -- a trait can exist without an ecology note (some traits are self-evident), but having the note strengthens the design.
