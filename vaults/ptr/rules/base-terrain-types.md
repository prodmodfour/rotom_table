Three foundational terrain types determine what can traverse them. Regular Terrain (dirt, short grass, cement, smooth rock, indoor) is freely walkable. Earth Terrain is underground with no existing tunnel — only Pokemon with the Burrower [[movement-traits|movement trait]] can shift through it. Underwater Terrain is submerged water — only Pokemon with the [[swimmer|Swimmer]] movement trait can move through it.

These base types exist alongside the modifier terrains ([[slow-terrain-doubles-movement]], [[rough-terrain-intervening-only]], [[blocking-terrain-impassable]]) in the [[multi-tag-terrain-system]]. The base type determines traversal permission; modifiers alter movement cost or accuracy.
