Status afflictions and injuries each add to the [[capture-rate-base-formula]]:

- Each Persistent condition (Burned, Frozen, Paralyzed, Poisoned, Sleep): +10
- Each Volatile condition: +5
- Each Injury: +5

Stuck (+10) and Slow (+5) are independent bonuses, not volatile conditions — see [[stuck-slow-separate-from-volatile]]. The rules use "Additionally" to introduce them as separate additions.
