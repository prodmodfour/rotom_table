The number of foes required to flank a target increases with the target's size: 2 for Small/Medium, 3 for Large, 4 for Huge, 5 for Gigantic. The flankers must be adjacent to the target but not adjacent to each other.

Large combatants occupying multiple squares count as multiple foes for flanking purposes, equal to the number of squares they occupy adjacent to the target. However, a single combatant can never flank alone regardless of squares occupied — minimum two combatants required.

Once flanked, the target takes the -2 evasion penalty from [[flanking-penalty-post-cap]].
