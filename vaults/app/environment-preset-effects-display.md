When an [[environment-preset-constants|environment preset]] is selected in the [[encounter-environment-section]], a description paragraph and a list of effect cards appear below the dropdown.

Each effect card has a typed icon and label:
- **Accuracy** — for accuracy penalty effects (shows penalty value and what negates it)
- **Terrain** — for terrain override effects (shows terrain rules)
- **Status** — for status trigger effects (shows condition applied on trigger)
- **Rule** — for custom/freeform rule text

Each card displays its description text and a "Dismiss this effect" button (X icon) that removes that individual effect from the active environment.

A "Clear Environment" button at the bottom removes the entire environment preset.

The Dim Cave preset shows two cards (Accuracy + Rule). The Dark Cave preset shows two cards (Accuracy + Rule). The Frozen Lake preset shows two cards (Terrain + Status). The Hazard Factory preset shows three cards (Rule + Rule + Rule).
