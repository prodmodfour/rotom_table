The Skills tab on the [[gm-pokemon-detail-page]] shows two areas: a skills list and a Training section.

The skills list shows the Pokemon's skill dice (e.g., "Athletics: 3d6"). When no skills are recorded, it displays "No skills recorded". Skills are clickable and trigger dice rolls via the [[pokemon-sheet-rolls-composable]].

The Training section displays:
- **Tutor Points** — a numeric value (e.g., 0)
- **Training EXP** — a numeric value (e.g., 0)

Tutor Points increase every 5 levels (1 + floor(level/5)).
