# Scene Canvas

The central area of the [[scene-editor-page]]. It displays pokemon sprites and character avatars at their stored positions. Each entity shows its sprite image and a name label below it.

Pokemon appear as their species sprite with the species name (or nickname) displayed beneath. Characters appear as circular avatars (with trainer sprites if assigned). All entities are positioned using [[scene-positions-use-percentages]].

Entities can be dragged to reposition them on the canvas. Groups appear as dashed-border rectangles with resize handles; dragging an entity onto a group assigns it to that group. Dragging a group moves its member sprites together.
