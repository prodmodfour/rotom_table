# Encounter Table Card

Each card on the [[encounter-tables-list-page]] shows:

- **Name** as a heading (e.g. "Bramblewick - General")
- **Description** paragraph
- **Levels** range (e.g. "3 - 10")
- **Density** badge (e.g. "Moderate")
- **Entries** count
- **Sub-habitats** count (only shown when the table has sub-habitats)
- **Top Pokemon** section — up to five species listed by name with their rarity label (e.g. "Common", "Uncommon"), followed by a "+N more" overflow indicator
- **Sub-habitats** names shown as tags at the bottom (e.g. "Night", "Day", "Castle Shadow")

Each card is a link to the table's detail URL. The entire card is clickable.
