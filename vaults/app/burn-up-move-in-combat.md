Burn Up appears in the [[encounter-act-modal-move-list]] as a red (Fire-type) button showing "Burn Up Fire DB 13 Daily x2 AC 2".

The [[encounter-move-target-panel]] displays:
- Class: Special, DB: 13, AC: 2, Range: Burst 1, Smite, SP.ATK: 80
- Effect text: "Until the end of the encounter, the user loses its Fire-Type (pure Fire-Types become Normal-Type)."

Burn Up is a high-power Burst 1 AoE with a dramatic self-cost: the user permanently loses its Fire typing for the rest of the encounter. DB 13 ties it with some of the highest damage bases observed, but Daily x2 frequency and the type loss make it a last-resort attack. The Smite keyword means it can hit through certain protections. This is the first observed move where the user's type changes as a consequence of using it.
