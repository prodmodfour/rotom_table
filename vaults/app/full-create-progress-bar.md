# Full Create Progress Bar

A horizontal strip of section indicators at the top of [[full-create-mode]]. Each indicator shows an icon, a section name, and a brief status summary.

The indicators and their default statuses are:
- Basic Info — "Name required"
- Background & Skills — "No background"
- Edges — "0/4"
- Classes & Features — "0 classes, 0 features"
- Combat Stats — "0/10 points"
- Biography — "Optional"

These summaries update as the form is filled in.
