Copycat appears in the [[encounter-act-modal-move-list]] as a grey (Normal-type) button showing "Copycat Normal Scene x2". No DB or AC values are displayed since it is a Status move.

The [[encounter-move-target-panel]] displays:
- Class: Status, Range: 4, 1 Target
- A "STAB" badge appears next to the type badge, which is unusual for a Status move
- Effect text: "Use the Move the target has used on their last turn. You may choose new targets for the Move. Copycat cannot miss."

Copycat duplicates whatever move the target used on its last turn, allowing the user to redirect it with new targets. The "cannot miss" property means both the Copycat itself and presumably the copied move bypass accuracy checks. The 4-meter range requires proximity to the target being copied. Scene x2 frequency limits it to two uses per encounter. This is a versatile move that adapts to the opponent's strategy — powerful moves used against the user can be turned back.
