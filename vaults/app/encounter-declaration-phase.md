# Encounter Declaration Phase

The first phase of each round in the [[encounter-combat-flow]]. A panel appears with:

- A phase icon and "Declaration Phase" label
- A combatant counter (e.g. "1 of 1")
- The current declarer's **name** and **Speed** value

Below the combatant info:

- **Action Type** dropdown with options from [[encounter-action-types]]
- **Description** textbox (placeholder: "Describe the declared action...")
- **Declare & Next** button — disabled until both an action type is selected and a description is entered

Declarations proceed from lowest speed to highest. Once all trainers have declared, the declared actions appear in the [[encounter-trainer-declarations-panel]].
