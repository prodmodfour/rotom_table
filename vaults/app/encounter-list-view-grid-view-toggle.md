# Encounter List View / Grid View Toggle

Two adjacent buttons ("List View" and "Grid View") on the GM encounter page above the combatant area. The active view is highlighted.

- **List View** — shows combatants as [[encounter-combatant-card]]s grouped into [[encounter-participant-sections]] (Players, Allies, Enemies) with inline HP controls, status buttons, and action buttons. Wild enemy Pokemon cards also display a [[capture-section-on-wild-pokemon-cards]] when a trainer is present
- **Grid View** — replaces the combatant list with the [[battle-grid]] VTT, showing tokens on a spatial grid alongside a [[encounter-combat-log]] sidebar
