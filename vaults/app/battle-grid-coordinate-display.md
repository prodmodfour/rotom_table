# Battle Grid Coordinate Display

A text label in the [[battle-grid]] area that shows the current cursor position and active tool. When a [[battle-grid-measurement-toolbar|measurement tool]] is active and a token is selected, it reads as coordinates followed by the tool name (e.g. "15, 1 | distance").

In isometric mode, the display also shows the Z elevation level at the hovered cell ([[elevation-cost-charges-per-level-change]]).
