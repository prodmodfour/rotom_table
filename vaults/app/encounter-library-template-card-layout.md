# Encounter Library Template Card Layout

Each template on the [[encounter-library-page]] is rendered as a card showing:

- **Header row** — template name (h3) on the left, [[encounter-template-card-actions]] icons (Load, Duplicate, Edit, Delete) on the right
- **Description** — source text if present (e.g. "Created from encounter: Gym Battle Template Source")
- **Meta row** — three labeled fields: Type (e.g. "Trainer Battle" or "wild"), Combatants count, Grid dimensions (e.g. "20x15")
- **Combatants list** — up to 5 combatants shown with a color-coded side dot (green=player, blue=ally, red=enemy), name with level, and type label ("pokemon"). If more than 5 exist, a "+N more" indicator appears.
- **Category badge** — a colored badge in the corner when a category is assigned (e.g. "Wild Battle" in scarlet)
- **Tags** — teal-colored tag chips when tags exist

Cards are displayed in a responsive grid layout.
