# Player View HP Formula Tooltip

The HP Base cell in the [[player-view-stats-section]] has a tooltip showing the Max HP calculation formula. It reads:

"Max HP = Level (x) x2 + HP Base (y) x3 + 10 = total"

For example, for a Level 1 character with HP Base 11: "Max HP = Level (1) x2 + HP Base (11) x3 + 10 = 45".
