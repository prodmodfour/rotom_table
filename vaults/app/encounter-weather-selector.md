# Encounter Weather Selector

A dropdown in the [[encounter-toolbar]] for setting the game weather condition. The tooltip reads "Set game Weather Condition (PTU p.341) — has mechanical combat effects. Not the same as natural weather."

Options: No Weather Condition (default), Sunny, Rain, Sandstorm, Hail, Snow, Fog, Harsh Sunlight, Heavy Rain, Strong Winds.
