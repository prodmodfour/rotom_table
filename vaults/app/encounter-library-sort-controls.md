# Encounter Library Sort Controls

The [[encounter-library-page]] has a sort dropdown and direction toggle button next to the search and category filters.

The sort dropdown offers three options:

- **Recently Updated** (default) — orders by last modification date
- **Recently Created** — orders by creation date
- **Name** — orders alphabetically

The direction toggle button displays an arrow: "↓" for descending (default) and "↑" for ascending. Clicking it flips the direction. For example, sorting by Name descending shows Z-A, while ascending shows A-Z.

The template count in the header (e.g. "8 templates") shows the total count regardless of active filters or search.
