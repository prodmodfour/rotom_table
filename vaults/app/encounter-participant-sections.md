# Encounter Participant Sections

In List View, the encounter page organizes combatants into three columns: **Players**, **Allies**, and **Enemies**. Each section has a heading and a "+ Add" button that opens the [[encounter-add-combatant-modal]].

Empty sections display placeholder text ("No players added", "No allies", "No enemies"). When populated, each combatant appears as an [[encounter-combatant-card]].
