# Player View Equipment Section

A collapsible section in the [[player-view-character-tab]], collapsed by default. The header reads "Equipment" with a chevron icon.

When expanded, it shows six equipment slots in a vertical list:

1. **Head**
2. **Body**
3. **Main Hand**
4. **Off Hand**
5. **Feet**
6. **Accessory**

Each slot shows the slot name on the left and the equipped item name on the right, or "-- Empty --" if nothing is equipped. The six slots correspond to the [[equipment-slot-definitions]].
