# Encounter Template Duplicate Is Instant

Clicking "Duplicate" on a template card in the [[encounter-library-page]] immediately creates a copy with no confirmation dialog or modal.

The duplicate's name is the original name with " (Copy)" appended. All other data — description, battle type, combatants, grid config, category, and tags — is copied identically. The duplicate is a new independent template record.
