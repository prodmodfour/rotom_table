# Player View Skills Section

A collapsible section in the [[player-view-character-tab]], collapsed by default. The header reads "Skills" with a chevron icon.

When expanded, it lists all 16 trainer skills in a vertical list. Each row shows the skill name on the left and its rank name on the right. The skills listed are: Acrobatics, Athletics, Charm, Combat, Command, Focus, General Ed, Guile, Intimidate, Intuition, Medicine Ed, Occult Ed, Perception, Pokemon Ed, Stealth, Survival, Technology Ed.

Rank names include Pathetic, Untrained, Novice, Adept, and potentially higher ranks.
