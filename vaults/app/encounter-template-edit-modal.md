# Encounter Template Edit Modal

Clicking "Edit" on a template card in the [[encounter-library-page]] opens a modal titled "Edit Template" with the following editable fields:

- **Name** (required) — text input, pre-filled with the template name
- **Description** (optional) — text input, pre-filled with existing description
- **Battle Type** — dropdown with Trainer Battle and Full Contact options
- **Category** — combobox with autocomplete placeholder "e.g., Boss Fights"
- **Tags** — comma-separated text input with placeholder "e.g., gym, leader, fire"

The footer has Cancel and "Save Changes" buttons. The same form is reused for the [[encounter-template-create-modal]], with different title and submit button text.
