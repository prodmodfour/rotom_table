The Classes tab on the [[gm-character-detail-page]] displays three lists:

1. **Trainer Classes** — a list of the character's trainer classes (e.g., "Ace Trainer", "Elite Trainer")
2. **Features** — a list of the character's features (e.g., "Let Me Help You With That", "Capture Specialist", "Agility Training", "Inspired Training")
3. **Edges** — a list of the character's edges (e.g., "Instinctive Aptitude", "Traveler", "Basic Skills [Command]", "Basic Skills [Perception]")

Each section has its own heading. The lists are read-only in both view and edit modes.
