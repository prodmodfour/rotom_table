# Scene Weather Is Narrative

The weather dropdown in the [[scene-properties-panel]] is explicitly labeled "Natural Weather" and carries the note: "Narrative only. Game Weather Conditions with combat effects are set separately in encounters."

This reflects PTU Core p.341's distinction between natural weather (a sunny day) and the game Weather Condition (the Sunny weather effect with mechanical combat implications). When an encounter is created from a scene, the encounter's weather is set to null — the two are independent.
