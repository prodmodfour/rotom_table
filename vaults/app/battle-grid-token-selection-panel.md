# Battle Grid Token Selection Panel

A panel that appears below the [[battle-grid]] canvas when a [[battle-grid-token-sprites|token]] is selected. It shows:

- The Pokemon's name with a **×** close button
- **Position** — the token's grid coordinates (e.g. "Position: (15, 1)")
- **HP** — the token's current and max hit points (e.g. "HP: 35/35")

Closing the panel via the × button deselects the token.
