# Player View Encounter Player List

A section within the [[player-view-encounter-tab]], below the VTT map, under the heading "Players." Lists the player's own Pokemon that are participating in the encounter.

Each entry shows the Pokemon's sprite, name, type badge(s), and exact HP values (e.g. "43 / 43"). The HP display uses the same green/teal bar style as the [[player-view-pokemon-card]].
