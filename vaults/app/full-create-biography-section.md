# Full Create Biography Section

The sixth section of [[full-create-mode]]. It is collapsible — clicking the "Biography" heading toggles it open or closed.

When expanded, it contains:
- Age (number spinner)
- Gender (text field with "Any value (PTU allows any)" placeholder)
- Height (number spinner with "cm" unit label)
- Weight (number spinner with "kg" unit label)
- Background Story (textarea with "Character background and history..." placeholder)
- Personality (textarea with "Character personality traits..." placeholder)
- Goals (textarea with "Character goals and aspirations..." placeholder)
- Money (number spinner defaulting to 5000, with "P" currency unit and a label noting "default 5,000 for level 1, PTU p. 17")

After creation, these fields appear in the [[gm-character-detail-notes-tab]] and are editable via [[gm-character-detail-edit-mode]].
