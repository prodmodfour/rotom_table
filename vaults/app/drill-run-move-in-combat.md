Drill Run appears in the [[encounter-act-modal-move-list]] as a brown (Ground-type) button showing "Drill Run Ground DB 8 At-Will AC 3".

The [[encounter-move-target-panel]] displays:
- Class: Physical, DB: 8, AC: 3
- Range: Melee, 1 Target
- ATK: 80
- Effect text: "Drill Run is a Critical Hit on 18+."

Drill Run has an enhanced critical hit range — 18+ instead of the standard 20. This broadened crit threshold significantly increases expected damage over many uses. DB 8 and At-Will frequency make it a reliable workhorse, though AC 3 is slightly less accurate than the standard AC 2. The simple Melee, 1 Target range with no keywords keeps the move straightforward. Ground typing provides important coverage against Electric, Fire, Poison, Rock, and Steel types.
