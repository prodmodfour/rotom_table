After clicking the [[capture-throw-button]], the capture section replaces the throw button with an accuracy roll result. The display shows:

- **"Accuracy: [roll] (need [threshold]+)"** — for example, "Accuracy: 20 (need 10+)"
- A highlighted message for special rolls, such as **"Natural 20 -- Critical Hit!"** in green text

The accuracy check determines whether the ball actually hits the target. If it misses, no capture roll occurs. If it hits, the server performs the capture roll. A [[capture-attempt-new-attempt-button]] appears afterward to allow retrying.
