# Encounter Table Create Modal

A modal titled "Create Encounter Table" opened by the "+ New Table" button on the [[encounter-tables-list-page]] or the [[habitats-list-page]].

Fields:

- **Name** (required) — placeholder: "e.g., Glowlace Forest"
- **Description** — placeholder: "Describe this habitat..."
- **Min Level** — defaults to 1
- **Max Level** — defaults to 10

Buttons: "Cancel" and "Create".
