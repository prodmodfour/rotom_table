# Encounter Template Card Actions

Each template card on the [[encounter-library-page]] has four icon buttons in its header row: Load, Duplicate, Edit, and Delete.

- **Load** opens the [[encounter-library-load-confirmation-modal]].
- **Duplicate** immediately creates a copy with no confirmation dialog — see [[encounter-template-duplicate-is-instant]].
- **Edit** opens the [[encounter-template-edit-modal]].
- **Delete** opens the [[encounter-template-delete-confirmation-modal]].
