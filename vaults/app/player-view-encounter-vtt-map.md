# Player View Encounter VTT Map

An interactive grid map within the [[player-view-encounter-tab]]. It displays Pokemon token sprites placed on a square grid with visible gridlines.

The token whose turn it is has a red glow effect behind its sprite, making it visually distinct from other tokens. Clicking a token reveals its grid coordinates as a small label (e.g. "15, 1").

Below the grid are [[player-view-encounter-zoom-controls]] for adjusting the map view.
