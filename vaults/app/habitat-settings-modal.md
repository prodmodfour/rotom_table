# Habitat Settings Modal

A modal titled "Table Settings" opened by the Settings button on the [[habitat-detail-page]].

Fields:

- **Name** — text input, pre-filled with current name
- **Description** — textarea, pre-filled with current description
- **Min Level** / **Max Level** — number inputs
- **Population Density** — dropdown combining the density tier and its description (e.g. "Moderate -- Small group -- a pack or family unit"). The help text reads "Describes the habitat's population density (informational -- does not control spawn count)" — see [[population-density-label]].

Buttons: "Cancel" and "Save".
