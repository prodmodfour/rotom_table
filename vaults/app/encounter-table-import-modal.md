# Encounter Table Import Modal

A modal titled "Import Encounter Table" opened by the "Import" button on the [[encounter-tables-list-page]].

Contains a dashed-border drop zone with a folder icon and the text "Click or drag JSON file here."

Buttons: "Cancel" and "Import".
