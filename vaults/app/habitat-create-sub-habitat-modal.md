# Habitat Create Sub-habitat Modal

A modal titled "Create Sub-habitat" opened by the "+ Add Sub-habitat" button in the [[habitat-sub-habitats]] section.

Fields:

- **Name** (required) — placeholder: "e.g., Deep Canopy"
- **Description** — placeholder: "Describe this sub-habitat..."
- **Level Range Override** (optional) — min and max spinbuttons

The "Create" button is disabled until a name is entered. "Cancel" closes the modal without changes. The same fields appear in the [[habitat-edit-sub-habitat-modal]] for existing sub-habitats.
