# Player View Features and Edges Section

A collapsible section in the [[player-view-character-tab]], collapsed by default. The header reads "Features & Edges" with a chevron icon.

When expanded, it shows two sub-lists under separate headings:

- **Features** — a list of feature names (e.g. "Let Me Help You With That", "Capture Specialist", "Agility Training")
- **Edges** — a list of edge names (e.g. "Instinctive Aptitude", "Traveler", "Basic Skills [Command]")

Only names are shown; no descriptions or mechanical details are displayed.
