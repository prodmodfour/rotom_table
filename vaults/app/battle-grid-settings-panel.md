# Battle Grid Settings Panel

A collapsible panel toggled by the "Settings" button in the [[battle-grid]] header. When open, it displays:

- **Width (cells)** — a spinbutton for the grid width (e.g. 20)
- **Height (cells)** — a spinbutton for the grid height (e.g. 15)
- **Cell Size (px)** — a spinbutton for the pixel size of each cell (e.g. 40)
- **Isometric View** — a checkbox to toggle isometric perspective
- **Background Image** — an "Upload Image" button for a custom map background
- **Reset** and **Apply** buttons to confirm or discard changes
