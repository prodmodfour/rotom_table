# Encounter Environment Section

A section below the [[encounter-combat-log]] showing the current battlefield environment.

- **Environment** label with the current value (default: "None")
- **Environment Preset** dropdown with options: None (default), Dim Cave (Blindness), Dark Cave (Total Blindness), Frozen Lake, Hazard Factory, Custom...

Selecting a preset populates the environment with effects from the [[environment-preset-constants]]. The [[environment-preset-effects-display]] shows typed effect cards below the dropdown.
