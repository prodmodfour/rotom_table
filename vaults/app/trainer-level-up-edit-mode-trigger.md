The [[trainer-level-up-modal]] can be triggered by a second path besides XP awards. On the GM character detail page, entering edit mode and changing the trainer's level field triggers a watcher that detects the level increase and opens the level-up modal.

This provides a manual override for GMs who need to adjust a trainer's level directly (e.g., story-granted level-ups or corrections) without going through the [[gm-character-detail-xp-section|XP award flow]].
