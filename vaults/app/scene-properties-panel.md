# Scene Properties Panel

A collapsible right-side panel in the [[scene-editor-page]]. Contains four fields:

- **Location Name** — text input with placeholder "e.g., Viridian Forest"
- **Background Image URL** — text input with placeholder "https://..."
- **Description** — multiline text area with placeholder "Scene description..."
- **Natural Weather** — dropdown with options: None, Sunny, Rain, Sandstorm, Hail, Snow, Fog, Harsh Sunlight, Heavy Rain, Strong Winds

Below the weather dropdown is the note: "Narrative only. Game Weather Conditions with combat effects are set separately in encounters." This reflects the [[scene-weather-is-narrative]] distinction.
