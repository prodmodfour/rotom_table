# Pokemon Basic Info Section

The first section of [[pokemon-creation-tab]], headed "Basic Info". Contains:

- Species (required text field with "Pikachu" placeholder, backed by [[species-autocomplete-loads-all-on-mount]])
- Nickname (text field with "Optional" placeholder)
- Level (number spinner, defaults to 1)
- Gender (dropdown: Male, Female, Genderless; defaults to Genderless)
- Shiny (dropdown: No, Yes; defaults to No)
- Location (text field with "e.g., Route 1" placeholder)
