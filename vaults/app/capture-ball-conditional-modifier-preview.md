In the [[capture-ball-selector-dropdown]], balls with conditional modifiers show an inline indicator next to the base modifier. When the condition is **not met**, a gray "if..." hint appears with a tooltip icon. Hovering or focusing the tooltip reveals a description of what triggers the conditional bonus (e.g., "-20 if the target was baited into the encounter with food."). When the condition **is met**, the display changes to a green checkmark with the actual modifier value.
