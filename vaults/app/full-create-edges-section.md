# Full Create Edges Section

The third section of [[full-create-mode]]. Shows a heading "Edges" with a counter "0 / 4".

The section provides a text input ("Enter edge name..." placeholder) with an "Add Edge" button that is disabled until text is entered. There is also an "Add Skill Edge" button.

A validation message reads: "Level 1 trainers should have 4 edges (4 starting edges) (have 0)".

The same text input and "Add Skill Edge" pattern is reused in the [[trainer-level-up-edges-step]].
