# Scene Manager Page

The GM scene list at `/gm/scenes`. The heading reads "Scene Manager" with the subtitle "Create and manage narrative scenes for Group View." A "+ New Scene" button sits in the top-right corner.

Scenes appear as cards in a grid. Each card shows a clapperboard icon placeholder (or background image if set), the scene name, and a count of entities (pokemon icon with a number). Each card has an "Activate" button, an "Edit" link that navigates to the [[scene-editor-page]], and a delete button (trash icon).

When a scene is active, its card gains a green border and an "Active" badge in the top-right corner, and the "Activate" button changes to "Deactivate."
