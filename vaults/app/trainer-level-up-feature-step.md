The Features step of the [[trainer-level-up-modal]], shown when the new level grants features (odd levels 3+, or via [[trainer-level-up-milestone-budget-effects|milestone bonus features]]).

Headed "Features" with a counter (e.g., "0 / 1"). If the trainer has classes, a "Your Classes:" row shows them as tags to remind the GM which class feature lists are available.

A text input with "Enter feature name..." placeholder and an "Add" button allows free-text feature entry. The input disables when the feature budget is filled. Added features appear as removable tags.

Below the input, a tip reads: "Features should come from your owned class feature lists."

A collapsible "Current Features (N)" details section shows the character's existing features as muted tags, for reference while choosing new ones.
