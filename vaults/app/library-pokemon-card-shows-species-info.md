Each Pokemon card on the Sheets page (`/gm/sheets`) under the "Pokemon" section displays species-derived information:

- Species sprite image (resolved via [[pokemon-sprite-resolution]])
- Nickname as the card heading (e.g., "Bulbasaur 1")
- Origin label (e.g., "Wild")
- Species name below the heading (e.g., "Bulbasaur")
- Type badges (e.g., "Grass", "Poison")
- Level
- HP bar with current/max values

The card links to the [[gm-pokemon-detail-page]] at `/gm/pokemon/:id`. Wild-generated Pokemon show their auto-assigned nickname (species name + number), while manually created or captured Pokemon show their given nickname.
