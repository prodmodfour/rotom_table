The Equipment tab on the [[gm-character-detail-page]] presents six equipment slots, each with a dropdown selector:

1. **Head** — options include Helmet, Dark Vision Goggles, Gas Mask, Re-Breather, Custom
2. **Body** — options include Light Armor, Heavy Armor, Stealth Clothes, Custom
3. **Main Hand** — only Custom option available
4. **Off-Hand** — options include Light Shield, Heavy Shield, Custom
5. **Feet** — options include Running Shoes, Snow Boots, Jungle Boots, Custom
6. **Accessory** — options include Focus (Attack), Focus (Defense), Focus (Special Attack), Focus (Special Defense), Focus (Speed), Custom

Each slot defaults to "-- Empty --" and renders as a dropdown. Each slot label is accompanied by a Phosphor icon from [[equipment-slot-definitions]].

When an item is equipped in a slot, the dropdown is replaced by a static display of the item name and an "Unequip item" button (X icon). Clicking unequip sends a request to clear the slot and restores the dropdown. See [[equipment-tab-equipped-slot-display]].

Below the slots is a "Browse Full Catalog" button that opens the [[equipment-catalog-browser-modal]].

A [[equipment-tab-combat-bonuses-section]] at the bottom summarizes active equipment bonuses.
