# Player View Combat Info Section

A collapsible section in the [[player-view-character-tab]], expanded by default. The header reads "Combat Info" with a chevron icon.

When expanded, it shows six values in a two-row grid:

- Top row: **Phys Evasion**, **Spec Evasion**, **Spd Evasion**
- Bottom row: **AP**, **Injuries**, **Temp HP**

Each cell shows a label above and a numeric value below.
