# Encounter No Active State

When no encounter is running, the GM encounter page (`/gm`) shows a centered panel titled "No Active Encounter" with the subtitle "Start a new encounter or load an existing one."

The panel contains:

- **Encounter Name** text field (default placeholder: "Route 1 Wild Battle")
- **Battle Type** dropdown with options: Trainer Battle (default), Full Contact
- **Start New Encounter** button
- A divider with "or"
- **Load from Template** button — opens the [[encounter-load-from-template-modal]]
- **Browse Library** link that navigates to the [[encounter-library-page]]
