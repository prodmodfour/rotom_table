# Full Create Validation Summary

A "Validation Summary" section at the bottom of [[full-create-mode]], above the "Create Human" button. It aggregates all validation issues from the form into a single list.

Each entry is prefixed with its category (e.g., "stats", "skills", "edges", "features") followed by the validation message. For a blank level-1 form the messages are:

- stats: Level 1 trainers should allocate exactly 10 stat points (currently 0)
- skills: Background should set exactly 1 skill to Adept (found 0)
- skills: Background should set exactly 1 skill to Novice (found 0)
- skills: Background should set exactly 3 skills to Pathetic (found 0)
- edges: Level 1 trainers should have 4 edges (4 starting edges) (have 0)
- features: Level 1 trainers should have 5 features (4 + 1 Training) (have 0)
