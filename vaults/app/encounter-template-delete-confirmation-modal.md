# Encounter Template Delete Confirmation Modal

Clicking "Delete" on a template card in the [[encounter-library-page]] opens a confirmation modal titled "Delete Template."

The body reads: `Are you sure you want to delete "[template name]"?` with the template name in bold, followed by "This action cannot be undone."

The footer has Cancel and "Delete Template" buttons.
