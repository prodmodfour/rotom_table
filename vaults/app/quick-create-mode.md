# Quick Create Mode

The Quick Create mode for [[human-character-creation-tab]] is described as "Minimal NPC scaffolding". It presents a compact form with three sections:

1. A basic info area with Name (required), Character Type dropdown (Player Character or NPC, defaults to NPC), Level (number spinner, defaults to 1), Location (text field with "e.g., Mesagoza" placeholder), and a [[trainer-sprite-chooser]].
2. A "Raw Stats" section with six number spinners for HP (default 10), Attack (default 5), Defense (default 5), Sp. Attack (default 5), Sp. Defense (default 5), and Speed (default 5).
3. A "Notes" textarea with "Additional notes..." placeholder.
