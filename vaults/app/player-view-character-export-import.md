# Player View Character Export Import

Two side-by-side buttons below the [[player-view-character-identity-and-hp]]:

- **Export Character** — labeled "Export character data as JSON", with a download icon
- **Import Character** — labeled "Import character data from JSON", with an upload icon

These allow the player to save or load their character data as JSON files.
