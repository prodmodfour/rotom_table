The Classes step of the [[trainer-level-up-modal]], shown when the advancement crosses level 5 or 10. Headed "New Trainer Class" with the triggering level(s) displayed.

A "Current Classes (N/4)" section shows existing classes as muted tags and newly chosen classes as green-bordered removable tags. The [[trainer-class-catalog|max of 4 classes]] is enforced.

Below, an italic hint reads: "Optional: Choose a new trainer class for which you qualify."

A searchable class picker organizes all 38 classes by the six categories (Introductory, Battling Style, Specialist Team, Professional, Fighter, Supernatural). Each class button shows the class name, a "[Branch]" badge for branching classes, and associated skills in italics. The search filters by name, category, description, and associated skills.

Clicking a branching class (Type Ace, Stat Ace, Style Expert, Researcher) opens a specialization picker: a dropdown of available specializations with Confirm and Cancel buttons. Already-taken specializations are excluded from the dropdown.

When the class limit is reached, the picker is replaced with "Maximum classes reached (4/4)".
