Constrict appears in the [[encounter-act-modal-move-list]] as a grey (Normal-type) button showing "Constrict Normal DB 1 At-Will AC 2".

The [[encounter-move-target-panel]] displays:
- Class: Physical, DB: 1 → 3 (STAB), AC: 2, Range: Melee, 1 Target, ATK: 80
- A "STAB" badge appears next to the type badge
- Effect text: "Lower the target's Speed by -1 CS. Constrict may be used as a Swift Action against targets the user is Grappling and automatically hits when performed this way."

Constrict is a weak melee attack (DB 1, the lowest possible) whose value comes from its grapple synergy. When used against a grappled target, it can be used as a Swift Action and automatically hits, bypassing the normal accuracy check. The guaranteed Speed -1 CS debuff pairs well with grapple tactics, slowing the target's initiative and evasion. At-Will frequency allows repeated use to stack Speed debuffs.
