Bulk Up appears in the [[encounter-act-modal-move-list]] as a brown (Fighting-type) button showing "Bulk Up Fighting EOT". No DB or AC values are displayed on the button since it is a Status move.

The [[encounter-move-target-panel]] displays:
- Class: Status, Range: Self
- Effect text: "Raise the user's Attack and Defense by +1 CS each."

The target panel omits DB and AC fields for Status moves, showing only Class and Range. Bulk Up is a self-targeting buff that raises both offensive and defensive Physical stats simultaneously. EOT frequency allows repeated use to stack combat stages over multiple turns, similar to other self-buff moves like [[agility-move-in-combat]] and [[amnesia-move-in-combat]].
