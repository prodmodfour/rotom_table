Agility appears in the [[encounter-act-modal-move-list]] as a pink (Psychic-type) button showing "Agility PSYCHIC EOT". No DB or AC is displayed.

The [[encounter-move-target-panel]] displays:
- Class: Status, Range: Self
- Effect text: "Raise the user's Speed 2 Combat Stages."

Because the range is "Self", only the user is a valid target. The opposing combatant appears greyed out with "Out of range". The +2 Speed CS effect is described in the text but not automatically applied — the GM must use the [[encounter-combat-stages-modal]] to adjust the speed combat stage manually.
