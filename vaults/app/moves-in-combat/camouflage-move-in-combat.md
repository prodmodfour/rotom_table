Camouflage appears in the [[encounter-act-modal-move-list]] as a gray (Normal-type) button showing "Camouflage Normal EOT". No DB or AC values are shown. A STAB badge appears on the target panel header because the Move Tester is Normal-type.

The [[encounter-move-target-panel]] displays:
- Class: Status, Range: Self
- Effect text: "The user changes their Type to match the field. Pick one type from the following table. If two or more rows are relevant (such as because of Weather), pick one:"

The effect text references a table that maps terrain and weather conditions to types, but the table itself is not displayed in the target panel — only the introductory text appears. This means the GM must reference the PTU rulebook for the actual type mapping. Camouflage is one of the few observed moves that changes the user's type, similar to [[burn-up-move-in-combat]] which also alters typing but as a cost rather than a benefit.
