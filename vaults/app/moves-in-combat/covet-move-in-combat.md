Covet appears in the [[encounter-act-modal-move-list]] as a grey (Normal-type) button showing "Covet Normal DB 6 At-Will AC 2".

The [[encounter-move-target-panel]] displays:
- Class: Physical, DB: 6 → 8, AC: 2
- Range: Melee, 1 Target
- ATK: 80
- A "STAB" badge appears next to the type badge — the DB display shows the STAB-boosted value (6 → 8) inline
- Effect text: "Covet takes the target's Held Item or Accessory Slot Item and attaches it to Covet's user, if the user is not holding anything."

Covet combines a modest Physical attack with item theft. The DB 6 base is boosted to 8 via STAB on the Normal-type Move Tester, and the target panel shows this boost explicitly with an arrow notation. At-Will frequency makes it freely repeatable. The item theft is conditional — the user must not already be holding an item — which limits its utility after the first successful steal. Melee range requires adjacency.
