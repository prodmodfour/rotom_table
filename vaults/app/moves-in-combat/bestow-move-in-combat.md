Bestow appears in the [[encounter-act-modal-move-list]] as a beige (Normal-type) button showing "Bestow Normal At-Will".

The [[encounter-move-target-panel]] displays:
- Class: Status, Range: Melee, 1 Target
- No AC, DB, or attack stat fields
- STAB badge (Normal-type move used by Normal-type Move Tester)
- Effect text: "The user gives its held item to the target, unless the target is already holding an item. Using Bestow is a Swift Action."

Bestow is a Status move that transfers a held item to the target. The Swift Action keyword means it does not consume the user's standard action for the turn. Both the item transfer and Swift Action mechanics are described in effect text and managed manually by the GM. At-Will frequency.
