Conversion appears in the [[encounter-act-modal-move-list]] as a grey (Normal-type) button showing "Conversion Normal At-Will". No DB or AC values are displayed since it is a Status move with no accuracy check.

The [[encounter-move-target-panel]] displays:
- Class: Status, Range: Self
- A "STAB" badge appears next to the type badge, which is unusual for a Status move — STAB has no mechanical effect on moves that deal no damage
- Effect text: "The user becomes the elemental Type of their choice as long as they have a Move that is the same elemental Type until the end of the encounter. Replace all other Types."

Conversion allows the user to change its type to match any type it has a move for, lasting until the encounter ends. This is a powerful defensive and offensive tool: the user can gain resistances, immunities, or STAB bonuses by choosing the right type. At-Will frequency means the type can be changed freely each turn, adapting to the current threat. The restriction to types matching known moves limits the available options.
