Manipulate does not appear in the [[encounter-combat-maneuvers]] grid or anywhere else in the encounter UI. It exists only as a data entry in the moves seed file (moves.csv, Maneuver section).

Per the seed data, Manipulate is a Standard action, AC 2, range 6 (1 Target). It is a trainer-only maneuver with three sub-options: [[bon-mot-maneuver-in-combat]], [[flirt-maneuver-in-combat]], and [[terrorize-maneuver-in-combat]]. Each Manipulation can only be used once per Scene per target.

Since there is no Manipulate button in the app's maneuver grid, the GM must track this maneuver entirely outside the app's UI.
