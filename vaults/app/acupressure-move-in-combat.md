Acupressure appears in the [[encounter-act-modal-move-list]] as a beige (Normal-type) button showing "Acupressure NORMAL EOT AC 2". No DB is displayed because it has no damage base.

The [[encounter-move-target-panel]] displays:
- Class: Status, AC: 2, Range: Melee, 1 Target or Self
- STAB badge (when used by a Normal-type Pokemon)
- Effect text: "Roll 1d6. On a result of 1, raise the target's Attack by +2 CS. On a result of 2, raise the target's Defense by +2 CS..." (random stat boost)

Acupressure is a Status-class move that still has an AC value (2), so it requires an accuracy roll unlike most status moves. The range "1 Target or Self" makes both the user and other combatants valid targets. The random 1d6 roll and stat application are not automated — the GM handles them manually.
