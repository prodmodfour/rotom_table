Below the stat rows in the [[trainer-level-up-stat-allocation-step]], a derived stats section shows:

- Max HP with a before -> after display (e.g., "48 -> 53"), which updates live as stat points are allocated
- Physical Evasion, Special Evasion, and Speed Evasion as single values

This mirrors the [[derived-stats-display]] from character creation but uses an arrow notation to show the change from current to new values.
