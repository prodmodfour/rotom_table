The [[gm-character-library-page]] provides a filter bar with five controls:

- A text search box with placeholder "Search..."
- A type filter dropdown: All Types, Humans Only, Pokemon Only
- A role filter dropdown: All Characters, Players, NPCs
- An origin filter dropdown: All Origins, Manual, Wild, Captured, Template, Imported
- A sort dropdown: Sort by Name, Sort by Level
- A Reset button that clears all filters

All filters default to their "All" option. The sort defaults to Sort by Name.
