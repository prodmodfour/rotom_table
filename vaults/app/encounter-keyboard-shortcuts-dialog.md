# Encounter Keyboard Shortcuts Dialog

A modal opened by clicking the "Keyboard shortcuts (?)" button on the GM encounter page, or by pressing `?`. It lists shortcuts organized into four sections:

**General** — Ctrl+Z undo, Ctrl+Shift+Z or Ctrl+Y redo, ? to toggle help.

**Virtual Tabletop - Grid** — +/= zoom in, - zoom out, 0 reset view, arrow keys pan, scroll wheel zoom, middle-click drag to pan.

**Virtual Tabletop - Selection** — Shift+Click add/remove from selection, Shift+Drag marquee select multiple tokens, Escape clear selection or cancel, Delete remove selected token (GM only).

**Virtual Tabletop - Movement & Range** — W toggle movement range, M distance mode, B burst AoE, C cone AoE, L line AoE, R cycle AoE direction, [/] decrease/increase AoE size.

**Virtual Tabletop - GM Tools** — T terrain painting mode, F fog of war editing, V fog reveal tool, H fog hide tool, E fog explore tool, [/] decrease/increase brush size.
