All sections of the [[player-view-character-tab]] are entirely read-only. Stats, combat info, skills, features & edges, equipment, and inventory display data without edit buttons, inline editing, or mutation controls.

The only interactive elements are the collapsible section headers (which toggle visibility) and the [[player-view-character-export-import]] buttons. The player cannot modify their character's stats, equip items, or change any values from this view.

Character editing is only available through the [[gm-character-detail-edit-mode|GM character detail page]].
