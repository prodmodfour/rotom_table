# Encounter Header

The header area at the top of the active encounter view on the GM encounter page (`/gm`). It shows:

- The encounter name as an `h2` heading (e.g. "New Encounter")
- A **battle type badge** (e.g. "Trainer Battle" in a colored pill)
- A **round counter badge** (e.g. "Round 1", "Round 2")
- A **phase badge** indicating the current [[encounter-combat-flow]] phase: "Declaration (Low → High)", "Resolution (High → Low)", or "Pokemon Phase"

The header also contains the [[encounter-toolbar]].
