# Player View Character Selection Button

Each character in the [[player-view-character-selection]] list is a button showing:

- A circular letter avatar with the first letter of the character's name, on a pink/magenta background
- The character's name
- Trainer classes, if any (e.g. "Ace Trainer / Elite Trainer"), displayed as a slash-separated list
- Level (e.g. "Lv. 1")
- A small sprite of the character's first Pokemon on the right side, if the character has Pokemon

Characters without Pokemon show only the letter avatar and text. Characters without trainer classes omit that line.
