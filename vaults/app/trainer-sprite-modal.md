# Trainer Sprite Modal

An overlay that appears when clicking "Choose Sprite" in the [[trainer-sprite-chooser]]. The modal has a heading "Select Trainer Sprite" and a close button.

It contains a search field ("Search sprites..." placeholder) and a row of category filter buttons: All, Protagonists, Gym Leaders, Elite Four & Champions, Villains & Admins, Team Grunts, Generic Male, Generic Female, Specialists, Other.

Below the filters, a grid of sprite buttons is displayed. Each button shows a small trainer sprite image with the character's name below it. Visible sprites include well-known Pokemon trainers such as Red, Ethan, Lyra, Brendan, May, Lucas, Dawn, Hilbert, Hilda, Brock, Misty, Lt. Surge, Erika, Koga, Sabrina, Blaine, and many more.

Clicking a sprite button selects it for the character being created. The category list and sprite entries are sourced from the [[trainer-sprite-catalog-constant]].
