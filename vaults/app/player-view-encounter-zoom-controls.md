# Player View Encounter Zoom Controls

A control strip below the [[player-view-encounter-vtt-map]] grid. It contains:

- A **+** button to zoom in
- A percentage label showing the current zoom level (e.g. "100%")
- A **-** button to zoom out
- A **home** button (house icon) to reset the view
