Block appears in the [[encounter-act-modal-move-list]] as a beige (Normal-type) button showing "Block Normal At-Will AC 2". No DB value is displayed because it is a Status-class move.

The [[encounter-move-target-panel]] displays:
- Class: Status, AC: 2, Range: Melee, 1 Target
- STAB badge (Normal-type move used by Normal-type Move Tester)
- No DB field shown
- Effect text: "The target is Stuck and Trapped until the beginning of your next turn."

Block is a Status move that requires an accuracy roll (AC 2) despite having no damage. Unlike self-targeting Status moves like [[acid-armor-move-in-combat]], Block targets an opponent at Melee range. Both Target Dummy and Move Tester appear as valid targets. At-Will frequency means the user can attempt to lock down a target every turn.
