# Trainer Class Picker

A sub-section of [[full-create-classes-and-features-section]] headed "Trainer Classes" with a counter "0 / 4". Contains a search field ("Search classes..." placeholder) and a categorized list of trainer classes.

The categories are:
- **Introductory**: Ace Trainer, Capture Specialist, Commander, Coordinator, Hobbyist, Mentor
- **Battling Style**: Cheerleader, Duelist, Enduring Soul, Juggler, Rider, Taskmaster, Trickster
- **Specialist Team**: Stat Ace [Branch], Style Expert [Branch], Type Ace [Branch]
- **Professional**: Chef, Chronicler, Fashionista, Researcher [Branch], Survivalist
- **Fighter**: Athlete, Dancer, Hunter, Martial Artist, Musician, Provocateur, Rogue, Roughneck, Tumbler
- **Supernatural**: Aura Guardian, Channeler, Hex Maniac, Ninja, Oracle, Sage, Telekinetic, Telepath, Warper

Each class button shows the class name and its associated skill prerequisites (e.g., "Ace Trainer — Command", "Capture Specialist — Acrobatics, Athletics, Stealth, Survival, Perception, Guile"). Classes with "[Branch]" can be taken with different specializations.
