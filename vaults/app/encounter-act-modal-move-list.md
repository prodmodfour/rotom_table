The [[encounter-act-modal]] displays the combatant's moves under a "Moves" heading, above the Standard Actions section. Each move appears as a wide, clickable button color-coded by Pokemon type (e.g. tan/yellow for Normal, green for Ground, purple for Dragon).

Each move button shows two rows:
- **Top row**: move name (bold) and type label (e.g. "Normal", "Ground", "Dragon")
- **Bottom row**: DB value (if applicable), frequency (At-Will/EOT/Scene/etc.), AC value

Status moves that have no damage base omit the DB field entirely. Clicking a move button opens the [[encounter-move-target-panel]] for that move.

If the combatant has no moves, a "Struggle" fallback (DB 4, Typeless) appears in the Standard Actions section instead.
