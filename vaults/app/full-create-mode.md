# Full Create Mode

The Full Create mode for [[human-character-creation-tab]] is described as "PTU-compliant multi-section". It presents a multi-section form guided by a [[full-create-progress-bar]] at the top.

The sections are, in order:
1. [[full-create-basic-info-section]] — Name, type, level, location, sprite
2. [[full-create-background-and-skills-section]] — Background selection and skill ranks
3. [[full-create-edges-section]] — Edge selection
4. [[full-create-classes-and-features-section]] — Trainer classes and features
5. [[full-create-combat-stats-section]] — Stat point allocation and derived stats
6. [[full-create-biography-section]] — Collapsible biographical fields

Below all sections sits a Notes textarea and a [[full-create-validation-summary]], followed by a "Create Human" button.
