# Player View Character Tab

The default tab after selecting a character. It displays the selected character's sheet as a scrollable page of collapsible sections.

At the top is the [[player-view-character-identity-and-hp]], followed by [[player-view-character-export-import]] buttons.

Below are collapsible accordion sections in order:

1. [[player-view-stats-section]] — expanded by default
2. [[player-view-combat-info-section]] — expanded by default
3. [[player-view-skills-section]] — collapsed by default
4. [[player-view-features-edges-section]] — collapsed by default
5. [[player-view-equipment-section]] — collapsed by default
6. [[player-view-inventory-section]] — collapsed by default

The entire tab is [[player-character-tab-is-read-only|read-only]] — no section offers editing controls.
