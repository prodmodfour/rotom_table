Clicking the "Manage" button on the [[gm-character-library-page]] toggles manage mode. When active, the button appears in an active/pressed state.

Manage mode reveals a "Pokemon by Origin" summary panel showing counts by origin type (e.g., "wild: 77", "manual: 84", "template: 22").

Below the summary are bulk action buttons:
- "Archive Unowned Wild (77)" — archives all unowned wild Pokemon
- "Delete Unowned Wild (77)" — deletes all unowned wild Pokemon

The number in parentheses reflects the current count of unowned wild Pokemon.
