# Player View Stats Section

A collapsible section in the [[player-view-character-tab]], expanded by default. The header reads "Stats" with a chevron icon.

When expanded, it shows six stat values in a two-row grid:

- Top row: **HP Base**, **ATK**, **DEF**
- Bottom row: **SPA**, **SPD**, **SPE**

Each stat shows its label above and numeric value below. The HP Base cell has a [[player-view-hp-formula-tooltip]] accessible via its aria label.
