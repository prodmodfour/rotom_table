# Player View Bottom Navigation

A fixed navigation bar at the bottom of the player view with four tabs, each showing an icon and label:

1. **Character** — opens the [[player-view-character-tab]]
2. **Team** — opens the [[player-view-team-tab]]
3. **Encounter** — opens the [[player-view-encounter-tab]]
4. **Scene** — opens the [[player-view-scene-tab]]

The active tab is visually highlighted. The Encounter tab shows a small notification dot when an encounter is active.
