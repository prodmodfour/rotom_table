# Skills Display

A read-only list of all trainer skills shown in the [[full-create-background-and-skills-section]]. Skills are grouped under three categories:

**Body**: Acrobatics, Athletics, Combat, Intimidate, Stealth, Survival

**Mind**: General Ed, Medicine Ed, Occult Ed, Pokemon Ed, Technology Ed, Guile, Perception

**Spirit**: Charm, Command, Focus, Intuition

Each skill shows its name and current rank. All skills default to "Untrained". When a background is selected, some skills change rank according to the background's assignments.
