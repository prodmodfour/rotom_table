# Create Character Page

The `/gm/create` route presents a "Create Character" page accessible from the GM navigation bar. The page title reads "GM - Create Character". A "← Back to Sheets" link in the top-left returns to the [[gm-character-library-page]].

The page offers two top-level entity type selectors: [[human-character-creation-tab]] and [[pokemon-creation-tab]]. These appear as large styled buttons at the top of the form area.
