# Scene Add Panel

A collapsible right-side panel in the [[scene-editor-page]] titled "Add to Scene." Two tabs switch between Characters and Pokemon content.

**Characters tab**: Lists all characters in the system with their name and role (e.g., "player"). Each row has an add button to place the character into the scene. Characters already in the scene do not appear in this list.

**Pokemon tab**: Shows characters that own pokemon, with a count badge showing how many pokemon each character has. An expandable list reveals individual pokemon that can be added. Below the character-owned pokemon is an "Add Wild Pokemon" section with a species text input, a level number spinner (default 5), and an add button. Wild pokemon added this way are [[scene-pokemon-are-transient-references]].
