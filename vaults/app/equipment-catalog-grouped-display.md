Within the [[equipment-catalog-browser-modal]], items are grouped by equipment slot. Each group has a header showing the slot's Phosphor icon and label (e.g., the baseball cap icon and "Head"). Groups appear in the order defined by [[equipment-slot-definitions]]: Head, Body, Main Hand, Off-Hand, Feet, Accessory.

Each item card shows:
- **Name** and **cost** (formatted with locale-aware thousands separators and a "P" suffix for Poké currency)
- A **description** line
- **Bonus badges** — color-coded pills for the item's mechanical effects: DR (blue), Evasion (green), Speed CS (amber), stat bonus (violet), conditional effects (pink), and "Can Ready" (teal)

Empty groups (no items for that slot) are hidden. When no items match the current filters, a centered italic "No items match your search" empty state appears.
