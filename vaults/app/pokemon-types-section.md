# Pokemon Types Section

The second section of [[pokemon-creation-tab]], headed "Types". Contains two dropdowns:

- Primary Type (required): defaults to Normal. Options are all 18 Pokemon types: Normal, Fire, Water, Electric, Grass, Ice, Fighting, Poison, Ground, Flying, Psychic, Bug, Rock, Ghost, Dragon, Dark, Steel, Fairy.
- Secondary Type: defaults to "None". Same 18 type options plus "None".
