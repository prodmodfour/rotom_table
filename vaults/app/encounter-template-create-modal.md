# Encounter Template Create Modal

Clicking "+ New Template" on the [[encounter-library-page]] header opens a modal titled "Create Template." The form has the same fields as the [[encounter-template-edit-modal]]:

- **Name** (required) — text input, empty
- **Description** (optional) — text input, empty
- **Battle Type** — dropdown defaulting to Trainer Battle
- **Category** — combobox with autocomplete from existing categories
- **Tags** — comma-separated text input

The "Create Template" button is disabled until the Name field is filled. This creates an empty template (no combatants) that can later be loaded and populated.
