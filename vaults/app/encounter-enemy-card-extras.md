# Encounter Enemy Card Extras

Enemy Pokemon cards in the [[encounter-participant-sections]] display additional elements beyond the standard [[encounter-combatant-card]] layout:

- **Type badges** — colored pills showing the Pokemon's types (e.g. "Poison", "Dark")
- **Trainer assignment** — a "Trainer" dropdown that assigns which trainer is engaging this enemy; defaults to the first available trainer
- **Capture section** — the [[capture-section-on-wild-pokemon-cards]], including ball type selection, capture conditions checkboxes, capture rate display, and throw button
