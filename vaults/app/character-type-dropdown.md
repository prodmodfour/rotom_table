# Character Type Dropdown

A dropdown present in both [[quick-create-mode]] and [[full-create-basic-info-section]] for human character creation. It offers two options:

- Player Character
- NPC (selected by default)

This determines whether the created character is a player-controlled trainer or an NPC.
