Each Pokemon in the [[gm-character-library-page]] is displayed as a clickable card that links to the [[gm-pokemon-detail-page]].

The card shows:
- The Pokemon's sprite image (via [[pokemon-sprite-resolution]])
- The Pokemon's nickname as a heading
- The origin label (e.g., "Wild")
- The species name (e.g., "Barboach")
- Type badges (e.g., "Water", "Ground")
- Level (e.g., "Level 13")
- HP as current/max (e.g., "HP: 44/44")

Pokemon are grouped by location under collapsible section headers that show the location name and count.
