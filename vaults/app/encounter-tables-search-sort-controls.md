# Encounter Tables Search Sort Controls

A toolbar on the [[encounter-tables-list-page]] with four controls:

- **Search** textbox ("Search tables...") — filters cards by matching text in the table name and description
- **Sort** dropdown — options: "Sort by Name" (default), "Sort by Created", "Sort by Updated"
- **Order** dropdown — options: "Ascending" (default), "Descending"
- **Reset** button — restores all controls to defaults
