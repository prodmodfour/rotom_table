# Encounter Priority Actions Panel

A panel that appears between each phase in the [[encounter-combat-flow]]. The header says "Priority Actions Available" with the text "Between turns — declare Priority actions before proceeding."

It lists every combatant in the encounter with their name and initiative value (e.g. "Ash Init 5", "Alolan Grimer 1 Init 25"). Each combatant has three buttons:

- **Priority (Full Turn)** — spends the combatant's full next turn to act now
- **Limited** — a limited priority action
- **Advanced** — an advanced priority action

If a combatant has already used their priority for this phase, their Priority (Full Turn) button is disabled.

At the bottom, a **No Priority — Continue** button skips the interrupt and proceeds to the next phase.
